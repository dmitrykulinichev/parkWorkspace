# Профіль
> `page:profile` · `/profile`

Ваш персональний кабінет: керуйте особистими даними, налаштовуйте безпеку та підключення Telegram-бота.

## Вкладки

| Вкладка | За замовчуванням | Опис |
|---|---|---|
| [`tab:profile-general`](./tab_profile_general.md) | ✓ | Тут ви можете змінити своє ім'я та контактний email. |
| [`tab:profile-password`](./tab_profile_password.md) |  | Для зміни паролю введіть поточний пароль та новий пароль два рази для підтвердже |
| [`tab:profile-telegram`](./tab_profile_telegram.md) |  | Підключіть Telegram-бота для отримування миттєвих сповіщень про важливі події у  |
| [`tab:profile-notifications`](./tab_profile_notifications.md) |  | Налаштуйте, які саме сповіщення ви хочете отримувати на email та в Telegram. |
| [`tab:profile-access`](./tab_profile_access.md) |  | Перегляд ваших поточних ролей та дозволів у системі. Також тут можна переключити |

> Детальна документація кожної вкладки — у відповідних файлах.
