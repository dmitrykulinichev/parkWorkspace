# Профіль
> `page:profile` · `/profile`

Ваш персональний кабінет: керуйте особистими даними, налаштовуйте безпеку та підключення Telegram-бота.

## Вкладки

| Вкладка | За замовчуванням | Опис |
|---|---|---|
| [`tab:profile-general`](./general.md) | ✓ | Тут ви можете змінити своє ім'я та контактний email. |
| [`tab:profile-password`](./password.md) |  | Для зміни паролю введіть поточний пароль та новий пароль два рази для підтвердже |
| [`tab:profile-telegram`](./telegram.md) |  | Підключіть Telegram-бота для отримування миттєвих сповіщень про важливі події у  |
| [`tab:profile-notifications`](./notifications.md) |  | Налаштуйте, які саме сповіщення ви хочете отримувати на email та в Telegram. |
| [`tab:profile-access`](./access.md) |  | Перегляд ваших поточних ролей та дозволів у системі. Також тут можна переключити |

> Детальна документація кожної вкладки — у відповідних файлах.
