# Debug Tools (Інструменти розробника)
{{PAGE:sys_debug_tools}}
Внутрішня панель для відладки системи в реальному часі.
## Можливості
- **Скидання кешу:** Очищення Redis/File кешів.
- **Емуляція подій:** Тестування відправки Telegram повідомлень.
- **API Inspector:** Перегляд сирих JSON відповідей від серверів.
