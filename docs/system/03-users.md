# Системні користувачі

{{PAGE:sys_users_list}}

Управління персоналом всієї платформи та адміністраторами парків.

## Що можна зробити на цій сторінці?
1. **Реєструвати адмінів:** Створення акаунтів для команди підтримки.
2. **Переглядати статус:** Бачити, хто з користувачів зараз онлайн.
3. **Імперсонація:** Вхід від імені будь-якого користувача через {{MODAL:sys_user_impersonate}}.
