# Управління клієнтами (Парками)

{{PAGE:sys_parks_list}}

Інструментарій для адміністрування тенентів (окремих автопарків) на платформі Garage24.

## 🚀 Створення нового парку
Через {{MODAL:sys_park_create}} Super Admin реєструє нового клієнта:
- **Назва:** Офіційна назва парку.
- **Власник (Owner):** Email та ПІБ головного адміністратора.
- **Активація:** Після створення система автоматично надсилає {{EMAIL:email_welcome_owner}} з посиланням на встановлення пароля.

## 👥 Користувачі та Імперсонація
- **Керування штатом:** Super Admin може бачити список усіх менеджерів конкретного парку.
- **Вхід від імені (Impersonation):** Через {{MODAL:sys_user_impersonate}} адмін може увійти в систему як власник будь-якого парку. Це критично для надання технічної підтримки та відладки конфігурацій.

## 🔐 Ізоляція та Безпека
Система гарантує, що дані парку А ніколи не потраплять до парку Б завдяки `ParkScope`. Super Admin — єдина роль, яка може працювати без цього обмеження для виконання глобальних звітів.
