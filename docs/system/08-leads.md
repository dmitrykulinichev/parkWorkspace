# Заявки з лендінгу (Leads)

{{PAGE:sys_leads}}

Центр обробки запитів на підключення від потенційних клієнтів.

## Що можна зробити на цій сторінці?
1. **Переглядати нові ліди:** ПІБ, телефон та назва парку з форми на сайті.
2. **Змінювати статус:** Опрацьовано, Очікує, Відмова.
3. **Конвертувати:** Створення парку на основі даних ліда в один клік через {{MODAL:sys_park_create}}.
