# Системні регламенти ТО

{{PAGE:sys_maint_standards}}

Управління еталонними стандартами обслуговування автомобілів.

## 📋 Мастер-каталог
Garage24 надає клієнтам готову базу регламентів для різних типів авто:
- **Типізація:** "Бензин", "Дизель", "Електро", "Гібрид".
- **Імпорт:** Новий парк може в один клік скопіювати ці стандарти собі, замість того щоб налаштовувати все з нуля.

## 🏗 Керування еталонами
Super Admin визначає:
- Назву роботи.
- Оптимальний інтервал (км та дні).
- Пріоритет (наприклад, "Масло" — 15, "Гальма" — 5).

## 📐 Зв'язок з клієнтами
Якщо ми оновлюємо системний регламент (наприклад, збільшуємо інтервал за рекомендацією виробника), це не змінює дані в парках автоматично (щоб не порушити їхні внутрішні правила), але дає можливість розробникам підтримувати актуальність бази знань.
