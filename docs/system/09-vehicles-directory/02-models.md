# Моделі автомобілів
{{PAGE:sys_vehicle_models}}
Довідник конкретних моделей ТЗ, згрупованих за марками.
## Що можна зробити?
1. **Додати модель:** Наприклад, "Corolla" для марки "Toyota".
2. **Типізувати:** Вказати, чи є модель седаном, універсалом чи мінівеном.
3. **Естетика:** Призначення дефолтних зображень для моделей (в розробці).
