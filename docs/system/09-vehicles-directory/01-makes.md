# Марки та Моделі авто
{{PAGE:sys_vehicle_makes}}
Глобальний довідник транспортних засобів, доступний усім паркам.
## Що можна зробити?
1. **Створювати марки:** Додавання логотипів та назв брендів.
2. **Створювати моделі:** Прив'язка до марок з вказанням типу кузова за замовчуванням.
3. **Очищення даних:** Об'єднання дублікатів (наприклад, "VW" та "Volkswagen").
