# Функціональні можливості (Features)

{{PAGE:sys_features}}

Керування доступністю модулів системи для різних тарифних планів.

## Що можна зробити на цій сторінці?
1. **Вмикати/Вимикати модулі:** Наприклад, доступ до Uklon API або Telegram-бота.
2. **Призначати обмеження:** Встановлювати ліміти (наприклад, макс. кількість авто) для конкретних фіч.
3. **Керувати релізами:** Відкривати нові фічі тільки для обраних "бета-тестерів" (парків).
