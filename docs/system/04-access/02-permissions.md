# Пермішени (Дозволи)
{{PAGE:sys_permissions}}
Детальне керування доступом до кожної дії в системі.
## Що можна зробити?
1. **Створювати атомарні права:** Наприклад, `vehicles.delete` або `finance.export`.
2. **Групувати права:** Об'єднання технічних пермішенів у зрозумілі блоки для призначення ролям.
3. **Аудит прав:** Перевірка, які саме права зараз мають активні користувачі.
