# Ролі та Права (RBAC)

{{PAGE:sys_roles_list}}

Глобальне налаштування матриці доступу.

## Що можна зробити на цій сторінці?
1. **Створювати ролі:** Визначення набору прав для стандартних посад (Механік, Бухгалтер).
2. **Налаштовувати пермішени:** Деталізація доступу до окремих кнопок та модулів (View, Create, Edit, Delete).
3. **Синхронізувати:** Примусове оновлення прав у всіх парках після зміни глобальної ролі.
