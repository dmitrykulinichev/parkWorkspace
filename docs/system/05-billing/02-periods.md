# Білінг-періоди
{{PAGE:sys_billing_periods}}
Керування часовими рамками для виставлення рахунків.
## Що можна зробити?
1. **Закривати періоди:** Фіксація нарахувань за минулий місяць.
2. **Генерувати інвойси:** Автоматичне створення рахунків на основі активності авто.
3. **Коригувати сальдо:** Ручне внесення правок у баланс парку.
