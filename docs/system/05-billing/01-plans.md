# Біллінг та Тарифікація (SaaS)

Модуль для монетизації сервісу та контролю оплат клієнтами.

## 💳 Тарифні плани
{{PAGE:sys_plans_list}}
Кожен парк підключений до певного тарифу:
- **Base Fee:** Фіксована щомісячна сума.
- **Pay-as-you-go:** Оплата за кожне активне авто в системі.
- **Редагування:** Через {{MODAL:sys_plan_edit}} можна змінювати ціни та набір доступних модулів (Features).

## 📊 Знімки активності (Snapshots)
Система щоночі робить "знімок" стану кожного парку (`BillingVehicleDay`):
- Фіксується кількість авто в статусі "Active".
- На основі цих знімків в кінці місяця формується рахунок.
- Весь журнал нарахувань доступний у {{PAGE:sys_billing_logs}}.
