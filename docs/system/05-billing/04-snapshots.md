# Знімки авто (Snapshots)
{{PAGE:sys_billing_snapshots}}
Технічний реєстр активності флоту для розрахунку вартості SaaS.
## Що можна зробити?
1. **Переглянути добовий звіт:** Скільки авто в кожному парку були активні о 00:00.
2. **Перевірити статуси:** Чому конкретне авто потрапило (або не потрапило) в білінг.
