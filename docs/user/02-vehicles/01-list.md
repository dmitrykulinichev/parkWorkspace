# Автомобілі (Реєстр флоту)
{{PAGE:fleet_vehicle_list}}

Центральний вузол обліку транспортних засобів. 

## Що можна зробити на цій сторінці?
1. **Додати авто:** Через {{MODAL:fleet_vehicle_create}} реєструються VIN, номер, марка, модель та тип палива.
2. **Оновити пробіг:** Через {{MODAL:fleet_mileage_update}} фіксуються нові показники одометра. Система валідує, щоб пробіг не був меншим за попередній.
3. **Керувати статусами:** Переведення авто в "Ремонт", "Активний" або "Вільний".
4. **Тегування:** Призначення кольорових міток (наприклад, "Оренда", "Власний") для швидкої фільтрації.
5. **QR-коди:** Генерація публічних токенів для доступу водіїв до швидких звітів без логіну.
