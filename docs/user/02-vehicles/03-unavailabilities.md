## Що можна зробити на цій сторінці?
1. **Фіксувати простої:** Внесення періодів СТО, ДТП або відсутності водія.
2. **Аналізувати причини:** Типізація простоїв для розуміння слабких місць парку.
3. **Коригувати плани:** Система автоматично мінусує дні простою з фінансового плану авто, роблячи звітність за тиждень коректною.
