# Розрахунок зарплати (Payroll)
{{PAGE:fin_payroll}}

## Що можна зробити?
1. **Звірити вал:** Бачити дохід кожного водія за тиждень (Uklon + Ручні).
2. **Нарахувати виплати:** Система автоматично рахує % згідно з {{MODAL:fin_scheme_edit}}, додає бонуси та віднімає штрафи.
3. **Вирахувати готівку:** Автоматичне віднімання суми, яку водій вже отримав від клієнтів готівкою.
4. **Закрити період:** Фіксація виплат та формування звіту для бухгалтерії.
