# Реєстр вкладок сторінок (Tabs Registry)

Цей реєстр описує внутрішню навігацію сторінок (Tabs).
**Слаг:** `tab_[сторінка]_[назва]`

| Слаг (Slug) | Назва вкладки | Батьківська сторінка | Опис |
| :--- | :--- | :--- | :--- |
| `tab_reports_operational` | Оперативна статистика | `fleet_reports` | Щоденна активність авто та водіїв. |
| `tab_reports_performance` | Виконання нормативів | `fleet_reports` | KPI та плани по доходу. |
| `tab_reports_maintenance` | Тех. обслуговування | `fleet_reports` | Стан ТО всього парку. |
| `tab_uklon_reports_dash` | Дашборд Uklon | `uklon_reports` | Загальні цифри по агрегатору. |
| `tab_uklon_reports_drivers` | Рейтинг водіїв | `uklon_reports` | Оцінки та відгуки Uklon. |
| `tab_settings_general` | Загальні | `admin_settings` | Назва парку, логотип. |
| `tab_settings_contacts` | Контакти | `admin_settings` | Телефони та підтримка. |
| `tab_fleet_settings_tags` | Теги | `fleet_settings` | Керування мітками авто. |
