# Звіти про проблеми (Issue Reports)

{{SCRN:page_drv_issue_reports_main}}

> Сторінка дозволяє переглядати, створювати та обробляти звернення (звіти про проблеми) від водіїв щодо автомобілів чи іншого робочого контексту.

## Огляд

Звіти про проблеми є інструментом зворотного зв'язку водіїв. Сторінка дозволяє диспетчерам відстежувати стан проблем (нові, потребують інформації, прийняті рішення, задачі), переглядати деталі, видаляти неактуальні записи та приймати управлінські рішення (наприклад, створення задач у відповідь на звіт).

## Реєстр даних

| Колонка | Опис | Логіка / Сортування |
| :--- | :--- | :--- |
| **Опис** | Текстовий зміст проблеми | Сортування за текстом |
| **Автомобіль** | Автомобіль, якого стосується проблема | - |
| **Задача** | Зв'язок із системою задач (якщо створено) | Клік → відкриває {{MODAL:issue_task_info}} |
| **Статус** | Поточний стан звіту (Новий, Рішення прийнято тощо) | Сортування за статусом |
| **Повідомив водій** | Водій, який подав звіт | - |
| **Дата повідомлення** | Дата та час створення звіту | Сортування за датою |
| **Дії** | Контекстне меню | Прийняти рішення, Деталі, Видалити |

## Функціональні можливості

- **Створити звіт:** Відкриває {{MODAL:issue_report_form}}.
- **Прийняти рішення:** Кнопка "Прийняти рішення" (доступна для нових звітів) відкриває {{MODAL:issue_report_actions}}.
- **Переглянути деталі:** Відкриває {{MODAL:issue_report_details}}.
- **Переглянути рішення:** Якщо звіт оброблено, відкриває {{MODAL:issue_report_actions}} у режимі перегляду.
- **Видалення:** Кнопка видалення у меню або списку.
- **Фільтрація:** Пошук за описом та фільтри за статусом.

## Технічні особливості

> Цей розділ призначений для розробників.

- **Права доступу:** Потребує `issue_reports.view`.
- **Логіка рішень:** Статуси `new` та `needs_info` дозволяють перехід до прийняття рішення (створення задачі, відхилення тощо).
- **Візуалізація:** Протерміновані звіти підсвічуються червоним фоном (клас `bg-red-50`).
