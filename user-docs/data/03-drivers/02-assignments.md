# Журнал призначень

{{SCRN:page_ops_assignments_main}}

> Сторінка дозволяє переглядати історію призначення автомобілів водіям, створювати нові записи, редагувати деталі призначень, керувати файлами та видаляти помилкові записи.

## Огляд

Сторінка є реєстром операційної діяльності, що фіксує зв'язки "Водій — Автомобіль" у часі. Вона забезпечує простежуваність (auditing) використання автопарку, зберігає інформацію про того, хто зробив призначення, та надає швидкий доступ до файлів або Telegram-чатів, пов'язаних з конкретним призначенням.

## Реєстр даних

| Колонка | Опис | Логіка / Сортування |
| :--- | :--- | :--- |
| **Водій** | ПІБ та телефон водія | Клік на рядок → відкриває {{MODAL:ops_assignment_details}} |
| **Автомобіль** | Державний номер, марка та модель | - |
| **Призначення** | Дата/час призначення та користувач, що виконав дію | - |
| **Завершення** | Дата/час повернення авто та користувач, що виконав дію | - |
| **Статус** | Активне або завершене призначення | - |
| **Дії** | Кнопки управління | Перегляд файлів, посилання на Telegram, деталі, редагування, видалення |

### Стани інтерфейсу

| Стан | Умова відображення | Скріншот |
| :--- | :--- | :--- |
| Порожній | Призначень не знайдено (після фільтрів або порожня база) | {{SCRN:page_ops_assignments_empty}} |
| Мобільний | Ширина екрана < 768px | {{SCRN:page_ops_assignments_mobile}} |

## Функціональні можливості

- **Створення призначення:** Кнопка "Створити призначення" відкриває {{MODAL:ops_assignment_form}}.
- **Перегляд деталей:** Клік на рядок таблиці відкриває {{MODAL:ops_assignment_details}}.
- **Редагування:** Іконка олівця відкриває {{MODAL:ops_assignment_form}} для редагування існуючого запису.
- **Видалення:** Іконка кошика відкриває підтвердження видалення.
- **Робота з файлами:** Іконка файлу (якщо присутні) відкриває прикріплені документи.
- **Посилання на Telegram:** Іконка запуску відкриває відповідний чат у Telegram.
- **Фільтрація:** Кнопка фільтрів дозволяє шукати за водієм, авто або датою.

- **Права доступу:** Потребує `assignments.view`.
