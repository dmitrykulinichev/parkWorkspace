## Що можна зробити?
1. **Створити профіль:** Через {{MODAL:drivers_create}} вносяться ПІБ, телефон, контакти родичів та дані посвідчення водія.
2. **Активувати Telegram:** {{MODAL:drivers_tg_connect}} — видача токена для підключення водія до бота.
3. **Переглянути карму:** Статистика успішності: % виконаних поїздок, відхилення та скасування.
4. **Контролювати документи:** Система підсвічує водіїв, у яких закінчується термін дії прав.
