## Що можна зробити?
1. **Планувати зміни:** Drag & Drop призначення водіїв на слоти "День", "Ніч" або "Доба".
2. **Бронювати ТО:** Створення слотів типу "Maintenance", що блокує можливість призначити туди водія.
3. **Бачити Факт:** Відображення реальних виходів водіїв поверх плану для контролю дисципліни.
4. **Копіювати тиждень:** Масове заповнення графіку на основі попереднього періоду.
