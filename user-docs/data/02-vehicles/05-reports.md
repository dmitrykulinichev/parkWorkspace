# Звіти

{{SCRN:page_rep_reports_main}}

> Центральний аналітичний хаб для моніторингу технічного стану, операційної продуктивності та виконання нормативів автопарку.

## Огляд

Розділ "Звіти" призначений для комплексного аналізу діяльності автопарку. Він акумулює дані з модулів автомобілів, поїздок, технічного обслуговування та фінансів, надаючи керівнику парку інструменти для прийняття рішень на основі фактичних показників та прогнозів.

## Функціональні можливості

- **Перемикання звітів:** Швидка навігація між типами аналітики через систему вкладок.
- **Адаптивність:** Повноцінна робота зі звітами на мобільних пристроях.
- **Контроль наявності даних:** Автоматична перевірка наявності автомобілів перед завантаженням звітів.

## Технічні особливості

> Цей розділ призначений для розробників.

- **Синхронізація з URL:** Активна вкладка зберігається в URL-параметрі `?tab=`, що дозволяє ділитися прямими посиланнями на конкретні звіти.
