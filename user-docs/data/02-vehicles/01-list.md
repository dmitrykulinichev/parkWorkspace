# Автомобілі (Реєстр флоту)

{{PAGE:page_fleet_list_main}}

Центральний вузол обліку та операційного управління транспортними активами парку. Сторінка забезпечує повний життєвий цикл управління автомобілем: від завантаження в базу до моніторингу його поточного стану та призначення персоналу.

## Функціональні можливості

- **Реєстрація активів:** Індивідуальне додавання авто або пакетний імпорт через {{MODAL:fleet_vehicle_import}}.
- **Моніторинг пробігу:** Контроль актуальних показників одометра ({{MODAL:fleet_mileage_update}}).
- **Управління станами:** Візуалізація та зміна статусів (В роботі, Вільний, Ремонт, Недоступний).
- **HR-інтеграція:** Призначення водіїв на автомобілі ({{MODAL:ops_assignment_form}}).
- **Технічний облік:** Швидка генерація записів про ТО ({{MODAL:maint_record_form}}) та звітів про несправності.
- **Експорт:** Вивантаження повної бази автопарку у формат Excel (.xlsx).

### Специфічні стани інтерфейсу
- **Порожній гараж:** Відображається, якщо в парку немає жодного авто. {{PAGE:page_fleet_list_empty}}
- **Мобільний вигляд:** Таблиця трансформується у список карток. {{PAGE:page_fleet_list_mobile}}

## Операційне управління

### Глобальні дії (Header)
- **Додати авто:** Відкриває форму {{MODAL:fleet_vehicle_form}}.
- **Теги:** Налаштування словника міток у {{MODAL:fleet_tag_settings}}.
- **Фільтри:** Розширена фільтрація за параметрами ({{MODAL:fleet_filters}}).

### Дії з об'єктом (Inline)
1. **Пробіг:** Швидке оновлення через {{MODAL:fleet_mileage_update}}.
2. **Водій:** Призначення ({{MODAL:ops_assignment_form}}) або зняття водія.
3. **Сервіс:** Створення запису про ТО ({{MODAL:maint_record_form}}).
4. **QR-код:** Генерація публічного коду ({{MODAL:fleet_vehicle_qr}}).
5. **Трансфер:** Передача в інший парк ({{MODAL:fleet_vehicle_transfer}}).
6. **Видалити:** Видалення об'єкта ({{MODAL:fleet_vehicle_delete}}).

## Технічні особливості
- **Валідація:** Система блокує введення пробігу, меншого за попередній. При помилках форма підсвічується {{MODAL:modal_fleet_veh_valid}}.
- **Синхронізація:** Статуси авто оновлюються автоматично при зміні в модулях ТО або Графіку.
- **Адаптивність:** Повна підтримка тач-інтерфейсів для мобільної версії.
