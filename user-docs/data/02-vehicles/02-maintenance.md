# Технічне обслуговування
{{PAGE:maint_record_log}}

Система контролю за станом авто та планування сервісних робіт.

## Що можна зробити на цій сторінці?
1. **Записати заїзд на сервіс:** Через {{MODAL:fleet_maintenance_add}} створюється група робіт (заміна масла, ремонт ходової тощо).
2. **Контролювати витрати:** Внесення вартості запчастин та робіт, що автоматично створює фінансову транзакцію.
3. **Відстежувати регламенти:** Бачити, скільки кілометрів залишилося до наступного ТО на основі останнього зафіксованого пробігу.
4. **Зберігати акти:** Завантаження фото нарядів та чеків безпосередньо до запису про ремонт.
