# Журнал технічного обслуговування (ТО)

{{PAGE:page_maint_record_list}}

Централізований журнал обліку всіх сервісних подій автопарку: від регламентних робіт (заміна масла, фільтрів) до позапланових ремонтів та заміни запчастин.

## Функціональні можливості

- **Групування заїздів:** Система об'єднує декілька видів робіт (наприклад, ТО + заміна колодок) в один "Заїзд на сервіс", що дозволяє вести точний фінансовий та часовий облік.
- **Моніторинг пробігу:** Кожен запис автоматично оновлює поточний пробіг автомобіля.
- **Документальний супровід:** Можливість прикріплення фото чеків, актів виконаних робіт та інших документів до всього заїзду або окремих робіт.
- **Фінансовий контроль:** Облік вартості кожної операції та загальної суми витрат на сервіс.
- **Історія обслуговування:** Повний архів робіт по кожному автомобілю для аналізу зносу та прогнозування витрат.

## Журнал записів (Таблиця)

Кожен рядок таблиці представляє окремий заїзд на СТО або проведення робіт власними силами.

| Колонка | Опис |
| :--- | :--- |
| **Автомобіль** | Назва та номер ТЗ. |
| **Виконані роботи** | Набір кольорових чіпів з назвами робіт. Синій — ТО, Помаранчевий — Ремонт. |
| **Пробіг** | Показник одометра на момент проведення робіт. |
| **Дата виконання** | Календарна дата та час проведення робіт. |
| **Створив** | Користувач системи, який вніс дані в журнал. |
| **Файли** | Іконки прикріплених документів з можливістю швидкого перегляду. |
| **Вартість** | Загальна сума заїзду. |

## Керування записами

### Пошук та фільтрація
Ви можете швидко знайти потрібні записи за допомогою:
- **Текстового пошуку:** по назві авто або коментарю.
- **Розширених фільтри ({{MODAL:maint_filters}}):** за періодом, типом робіт, конкретним автомобілем або механіком.

### Дії
- **Додати запис ТО:** Відкриває універсальну форму {{MODAL:maint_record_form}}.
- **Деталі:** Повний перегляд структури заїзду у {{MODAL:maint_record_details}}.
- **Редагувати:** Зміна даних існуючого запису.
- **Видалити:** Повне видалення запису з системи (потребує підтвердження).

## Технічні особливості
- **Синхронізація з фінансами:** Кожен створений запис автоматично генерує фінансову транзакцію в категорії "Ремонт та ТО".
- **Автоматичне оновлення пробігу:** Система валідує пробіг у записі ТО та оновлює головний одометр авто, якщо вказане значення є більшим за поточне.
- **Адаптивність:** На мобільних пристроях таблиця перетворюється на стрічку подій, де зручно переглядати фото актів прямо "біля капота".
