# Документи автомобілів

{{PAGE:page_fleet_document_list}}

Централізований реєстр усіх юридичних та технічних документів автопарку. Система дозволяє не лише зберігати скан-копії, а й автоматично відстежувати терміни їх дії, надсилаючи сповіщення про необхідність оновлення.

## Функціональні можливості

- **Контроль термінів:** Автоматичне кольорове маркування документів, термін дії яких закінчується або вже вичерпаний.
- **Типізація:** Класифікація за категоріями: техпаспорт, ОСЦПВ, КАСКО, дозволи на таксі, договори оренди тощо.
- **Фінансовий зв'язок:** Можливість реєстрації витрат на оформлення документа безпосередньо у формі створення.
- **Цифровий архів:** Надійне зберігання файлів (PDF, JPG, PNG) з можливістю миттєвого перегляду в браузері.
- **Швидкий пошук:** Пошук за номером документа, автомобілем або типом.

## Реєстр документів (Таблиця)

| Колонка | Опис | Технічні деталі |
| :--- | :--- | :--- |
| **Автомобіль** | Назва та номер ТЗ, до якого належить документ. | |
| **Тип документа** | Назва категорії (напр. ОСЦПВ). | |
| **Номер** | Серія та номер паперового бланка. | |
| **Дата видачі** | День оформлення документа. | |
| **Термін дії** | Дата закінчення. | Підсвічується червоним, якщо документ прострочений. |
| **Статус** | Поточний стан (Активний / Прострочений). | |
| **Файли** | Іконки завантажених скан-копій. | |

## Керування

### Дії
- **Додати документ:** Відкриває форму {{MODAL:fleet_document_form}}.
- **Редагувати:** Зміна даних або завантаження нових файлів.
- **Видалити:** Повне видалення запису, пов'язаних файлів та фінансових транзакцій.
- **Фільтрація ({{MODAL:fleet_document_filters}}):** Пошук за конкретним авто або типом (напр. показати всі страховки).

## Технічні особливості
- **Синхронізація транзакцій:** При вказанні вартості документа система автоматично створює фінансовий запис у категорії "Адмін. витрати".
- **Аудит файлів:** Видалення документа автоматично очищує хмарне сховище від пов'язаних з ним медіа-файлів.
