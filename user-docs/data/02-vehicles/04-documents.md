# Документи

{{SCRN:page_docs_list_main}}

> Сторінка дозволяє переглядати, створювати та керувати документами, пов'язаними з транспортними засобами.

## Огляд

Розділ централізує облік усієї дозвільної та технічної документації автопарку (страховки, техпаспорти, дозволи). Сторінка інтегрована з модулем автопарку — кожен документ прив'язується до конкретного транспортного засобу.

## Реєстр даних

| Колонка | Опис | Логіка / Сортування |
| :--- | :--- | :--- |
| **Документ** | Назва або номер документа | Текстове поле |
| **Автомобіль** | Зв'язок з транспортним засобом (номер, модель) | Посилання на детальну картку авто |
| **Дата початку** | Дата початку дії документа | Сортування за часом |
| **Дата закінчення** | Дата закінчення дії документа | Колірна індикація терміну придатності |
| **Файл** | Посилання на завантажений скан/файл | Відкриває файл у новій вкладці |
| **Дії** | Кнопки управління записом | Редагування, видалення |

### Стани інтерфейсу

| Стан | Умова відображення | Скріншот |
| :--- | :--- | :--- |
| Порожній | Список документів порожній | {{SCRN:page_docs_list_empty}} |
| Мобільний | Ширина екрана < 768px | {{SCRN:page_docs_list_mobile}} |
| Прострочений | Дата закінчення < поточна дата | {{SCRN:page_docs_list_expired}} |

## Функціональні можливості

- **Додавання документа:** Створення нового запису з прикріпленням файлу. Використовує {{MODAL:fleet_document_form}}.
- **Редагування:** Оновлення даних наявного документа. Використовує {{MODAL:fleet_document_form}}.
- **Фільтрація:** Пошук документів за авто, типом, термінами дії. Використовує {{MODAL:fleet_document_filters}}.
- **Видалення:** Безповоротне видалення запису.

## Технічні особливості

- **Валідація:** Перевірка коректності дат (початок не може бути після закінчення) та обов'язковість завантаження файлу.
- **Права доступу:** Доступ до сторінки обмежений дозволом `document.view`.
- **Повідомлення:** Система автоматично підсвічує документи, термін дії яких добігає кінця (або вже закінчився).
