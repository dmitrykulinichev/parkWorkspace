# Документи автомобілів

{{PAGE:page_docs_list_main}}

Централізований реєстр усіх юридичних та технічних документів автопарку (техпаспорти, страховки, дозволи на таксі тощо). Система дозволяє надійно зберігати скан-копії та автоматично відстежувати терміни їх дії, запобігаючи штрафам та зупинкам ТЗ.

## Функціональні можливості

- **Контроль дедлайнів:** Автоматичне підсвічування документів, термін дії яких вичерпано або закінчується найближчим часом.
- **Цифровий архів:** Моментальний доступ до оригіналів файлів (PDF, фото) без необхідності пошуку в паперовій базі.
- **Динамічні поля:** Підтримка специфічних даних для різних типів документів (напр. сума покриття для ОСЦПВ).
- **Облік витрат:** Реєстрація фінансових витрат на оформлення документів з автоматичним відображенням у категоріях парку.
- **Контроль дублікатів:** Захист від створення декількох однотипних активних документів для одного автомобіля.

### Специфічні стани інтерфейсу
- **Порожній список:** Відображається, якщо в парку ще не додано жодного документа. {{PAGE:page_docs_list_empty}}
- **Мобільна версія:** Адаптоване відображення карток документів для смартфонів. {{PAGE:page_docs_list_mobile}}
- **Прострочені:** Вигляд рядків з активним попередженням. {{PAGE:page_docs_list_expired}}

## Операційне управління

### Глобальні дії
- **Додати документ:** Відкриває універсальну форму {{MODAL:modal_docs_form_main}}.
- **Пошук та фільтри:** Швидкий пошук за номером або фільтрація за типом/автомобілем ({{MODAL:modal_docs_filters}}).

### Дії з об'єктом
1. **Редагувати:** Зміна даних або заміна файлів.
2. **Переглянути:** Швидкий доступ до прикріплених зображень.
3. **Видалити:** Повне видалення документа та пов'язаних витрат.

## Технічна інформація (для розробників)

| Файл | Роль |
| :--- | :--- |
| `resources/js/pages/Documents/index.jsx` | Головний компонент сторінки. |
| `resources/js/hooks/useDocumentsPage.js` | Управління станом списку, пагінацією та фільтрами. |
| `resources/js/queries/documents.js` | CRUD мутації та запити React Query. |

## Технічні особливості
- **Адаптивна форма:** Набір полів у {{MODAL:modal_docs_form_dyn}} автоматично змінюється залежно від обраного типу документа.
- **Валідація конфліктів:** Якщо для авто вже є активний документ цього типу, система вимагатиме підтвердження ({{MODAL:modal_docs_form_force}}).
- **Синхронізація транзакцій:** Поле "Вартість" автоматично генерує фінансовий запис у базі даних.
