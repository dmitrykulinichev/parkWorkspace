# Глобальні налаштування

{{SCRN:page_settings_main}}

> Сторінка дозволяє налаштовувати параметри зовнішнього вигляду системи, зокрема перемикати колірну тему.

## Огляд

На цій сторінці зосереджені глобальні параметри інтерфейсу користувача. Наразі доступне управління колірною темою системи (світла/темна).

## Вкладки

### Зовнішній вигляд

{{SCRN:tab_settings_appearance}}

> Налаштування теми оформлення та персоналізації інтерфейсу.

#### Функціональні можливості

- **Тема оформлення:** Перемикання між світлою та темною темами (через `ThemeToggle`).

#### Управління (Actions)

| Дія | Результат |
| :--- | :--- |
| **Зберегти налаштування** | Фіксує вибрані параметри зовнішнього вигляду в системі. |

## Технічна інформація

> Цей розділ призначений для розробників.

| Файл | Роль |
| :--- | :--- |
| `resources/js/pages/Settings/index.jsx` | Головний компонент налаштувань |
| `resources/js/pages/Settings/AppearanceTab.jsx` | Вкладка зовнішнього вигляду |
| `resources/js/hooks/useSettings.js` | Хук для роботи з налаштуваннями |
| `resources/js/components/UI/ThemeToggle.jsx` | Компонент перемикання теми |
