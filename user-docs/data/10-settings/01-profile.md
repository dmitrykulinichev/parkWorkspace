# Профіль

{{SCRN:page_profile_main}}

> Сторінка дозволяє керувати особистими даними користувача, налаштуваннями безпеки та доступом до різних парків у межах системи.

## Огляд

Сторінка надає централізований інтерфейс для управління профілем користувача. Вона розділена на вкладки, що дозволяють редагувати особисту інформацію, змінювати пароль, налаштовувати сповіщення, Telegram-інтеграцію та переглядати рівні доступу. Налаштування є глобальними для всіх парків користувача.

## Вкладки

### Загальні дані

{{SCRN:tab_profile_general}}

> Редагування імені та email користувача.

#### Функціональні можливості

- **Оновлення даних:** Збереження імені та email у профілі.
- **Інформація:** Відображення дати реєстрації та останнього оновлення профілю.

#### Управління (Actions)

| Дія | Результат |
| :--- | :--- |
| **Зберегти зміни** | Оновлює особисті дані на сервері. |

### Зміна паролю

{{SCRN:tab_profile_password}}

> Зміна поточного пароля доступу до системи.

#### Функціональні можливості

- **Безпека:** Вимагає поточний пароль для підтвердження змін.

#### Управління (Actions)

| Дія | Результат |
| :--- | :--- |
| **Змінити пароль** | Оновлює пароль в системі. |

### Telegram

{{SCRN:tab_profile_telegram}}

> Налаштування інтеграції з Telegram для отримання сповіщень.

#### Функціональні можливості

- **Статус підключення:** Перегляд поточного статусу.
- **Генерація QR-коду:** Створення коду для підключення акаунту до Telegram-бота.
- **Відключення:** Скасування зв'язку з Telegram.

#### Управління (Actions)

| Дія | Результат |
| :--- | :--- |
| **Згенерувати код** | Створює нове посилання та QR-код для підключення. |
| **Відключити** | Розірвання зв'язку з Telegram. |

### Розсилки

{{SCRN:tab_profile_notifications}}

> Управління налаштуваннями регулярних звітів.

#### Функціональні можливості

- **Вибір каналів:** Включення/вимикання доставки звітів через Email або Telegram для кожного типу звіту окремо.
- **Ручне відправлення:** Можливість миттєво надіслати звіт.

#### Управління (Actions)

| Дія | Результат |
| :--- | :--- |
| **Зберегти налаштування** | Оновлює конфігурацію розсилок. |
| **Надіслати (Звіт)** | Миттєва відправка вибраного звіту. |

### Доступ та Безпека

{{SCRN:tab_profile_access}}

> Перегляд призначених ролей, дозволів та перемикання між доступними парками.

#### Реєстр даних

| Колонка | Опис |
| :--- | :--- |
| **Ролі** | Chip-список призначених ролей. |
| **Дозволи** | Список всіх активних дозволів у системі. |
| **Парки** | Список доступних для перемикання парків (лише для мульти-парк акаунтів). |

#### Управління (Actions)

| Дія | Результат |
| :--- | :--- |
| **Перейти (Парк)** | Перемикає контекст роботи на вибраний парк. |
