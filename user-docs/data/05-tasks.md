# Задачі

{{SCRN:page_tasks_main}}

> Сторінка призначена для управління списком задач, відстеження їх статусу та пріоритету.

## Огляд

Централізований інтерфейс для роботи з поточними справами в системі. Задачі можуть бути пов'язані з різними модулями (наприклад, технічне обслуговування, адміністративні справи).

## Функціональні можливості

- **Створення задачі:** Відкриває {{MODAL:task_form}}.
- **Редагування:** Редагування існуючої задачі через {{MODAL:task_form}}.
- **Перегляд деталей:** Детальна інформація через {{MODAL:task_details}}.
- **Зміна статусу:** Швидка зміна стану виконання.
- **Фільтрація:** Пошук та фільтрація задач за статусом, пріоритетом.

## Реєстр даних

| Колонка | Опис | Логіка / Сортування |
| :--- | :--- | :--- |
| **Заголовок** | Назва задачі | Клік -> деталі |
| **Статус** | Поточний стан | Колірне маркування |
| **Пріоритет** | Рівень важливості | Сортування за пріоритетом |
