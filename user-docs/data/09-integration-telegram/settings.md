# Налаштування Telegram

{{SCRN:page_intg_tele_settings_main}}

> Сторінка призначена для підключення та налаштування Telegram-бота, який забезпечує автоматизацію сповіщень та взаємодію з системою через Telegram-групи.

## Огляд

Сторінка дозволяє налаштувати інтеграцію з Telegram-ботом, встановити Webhook для отримання повідомлень, керувати підключеними Telegram-групами та їх функціоналом. Налаштування інтеграції є центральним елементом для отримання системних сповіщень у реальному часі.

## Функціональні можливості

- **Статус бота:** Глобальне увімкнення/вимкнення бота в системі.
- **Bot Token:** Введення токена API Telegram для автентифікації бота.
- **Webhook:** Встановлення Webhook URL для отримання оновлень від сервера Telegram.
- **Глобальні функції:** Керування списком доступних функцій (команд) бота для всієї системи.
- **Управління групами:** Додавання, редагування та видалення Telegram-груп, для яких налаштовується поведінка бота.
- **Тестування:** Можливість перевірити з'єднання з API Telegram та статус Webhook.

## Управління (Actions)

### Глобальні дії

| Дія | Результат |
| :--- | :--- |
| **Статус бота (тумблер)** | Увімкнення або вимкнення бота. |
| **Встановити Webhook** | Відправляє запит до Telegram для встановлення системного Webhook URL. |
| **Зберегти налаштування** | Зберігає всі поточні конфігурації бота, груп та функцій у базі даних. |
| **Видалити** | Видаляє всі налаштування інтеграції з Telegram. |
| **Перевірити з'єднання** | Виконує тест з'єднання з Telegram API та аналізує статус Webhook. |

### Дії з групами

| Дія | Результат |
| :--- | :--- |
| **Додати групу** | Створює нове поле для конфігурації додаткової групи. |
| **Видалити групу** | Видаляє групу з конфігурації після підтвердження. |

## Технічні особливості

- **Валідація:** Bot Token повинен бути дійсним токеном Telegram API.
- **Синхронізація:** При збереженні налаштувань автоматично оновлюється глобальний статус `listen_to_channel`.
- **Group Privacy:** Для коректної роботи бота в групах необхідно вимкнути "Group Privacy" в налаштуваннях бота через @BotFather.
- **Webhook статус:** Відображається інформація про кількість повідомлень, що очікують, та остання помилка, отримані від Telegram API.
