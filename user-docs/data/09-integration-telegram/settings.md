# Налаштування Telegram

{{SCRN:page_intg_telegram_main}}

> Сторінка дозволяє керувати інтеграцією системи з Telegram-ботом, налаштовувати Webhook та параметри підключених груп.

## Огляд

На цій сторінці адміністратор може підключити Telegram-бота до системи, налаштувати вебхук для отримання оновлень та керувати правами доступу груп (наприклад, дозволяти надсилання сповіщень чи обробку команд). Сторінка тісно інтегрована з сервісом інтеграцій системи.

## Функціональні можливості

- **Активація бота:** Глобальне увімкнення/вимкнення бота.
- **Bot Token Management:** Безпечне введення та приховування токена бота.
- **Webhook Management:** Встановлення та моніторинг статусу вебхука для отримання повідомлень.
- **Управління групами:** Додавання та налаштування декількох Telegram-груп, для кожної з яких можна визначити доступні команди.
- **Тестування з'єднання:** Інструмент для перевірки зв'язку з Telegram API та стану вебхука.

## Управління (Actions)

### Глобальні дії

| Дія | Результат |
| :--- | :--- |
| **Зберегти налаштування** | Оновлює конфігурацію бота та груп на сервері. |
| **Встановити Webhook** | Відправляє запит до Telegram API для реєстрації URL-адреси вебхука системи. |
| **Перевірити з'єднання** | Запускає діагностику API бота, вебхука та тест групи. |
| **Видалити** | Видаляє всі налаштування інтеграції з системи. |

### Дії з групами

| Дія | Результат |
| :--- | :--- |
| **Додати групу** | Створює новий запис для налаштування групи. |
| **Видалити групу** | Видаляє групу зі списку активних підключень. |

## Технічна інформація

> Цей розділ призначений для розробників.

| Файл | Роль |
| :--- | :--- |
| `resources/js/pages/TelegramSettings/index.jsx` | Головний компонент сторінки. |
| `resources/js/components/Settings/TelegramIntegrationSettings.jsx` | Компонент налаштувань. |
| `resources/js/services/api.js` | API-сервіс для запитів до налаштувань. |

## Технічні особливості

- **Webhook:** Бот потребує встановленого вебхука для роботи в реальному часі.
- **Group Privacy:** Для роботи бота в групах необхідно вимкнути "Group Privacy" у BotFather.
- **Тестування:** Модальне вікно тестування відображає статуси підключення до API, вебхука та каналу/групи.
