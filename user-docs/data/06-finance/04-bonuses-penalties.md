# Премії та Штрафи

{{SCRN:page_bonuses_penalties_main}}

> Сторінка призначена для фінансового контролю діяльності водіїв через систему премій та штрафів.

## Огляд

Сторінка забезпечує реєстр усіх нарахованих премій та штрафів водіям. Вона дозволяє додавати нові записи, редагувати існуючі та видаляти їх, а також фільтрувати записи за різними критеріями для зручного аналізу фінансових показників водіїв.

## Реєстр даних

| Колонка | Опис | Логіка / Сортування |
| :--- | :--- | :--- |
| **Тип** | Премія або Штраф | Візуальне маркування кольором |
| **Дата** | Дата події | Сортування за замовчуванням |
| **Водій** | ПІБ водія | Клік відкриває {{MODAL:drv_driver_details}} |
| **Сума** | Грошова сума (грн) | Кольорове виділення (+/-) |
| **Автомобіль** | Прив'язаний автомобіль | Клік відкриває {{MODAL:fleet_vehicle_details}} |
| **Дії** | Кнопки управління | Перегляд, Редагування, Видалення |

### Стани інтерфейсу

| Стан | Умова відображення | Скріншот |

## Функціональні можливості

- **Додавання запису:** Створення нової премії або штрафу. Використовує {{MODAL:bonus_penalty_form}}.
- **Редагування запису:** Зміна існуючих даних. Використовує {{MODAL:bonus_penalty_form}}.
- **Детальний перегляд:** Перегляд повної інформації по запису. Використовує {{MODAL:bonus_penalty_details}}.
- **Видалення:** Видалення помилкового або неактуального запису.
