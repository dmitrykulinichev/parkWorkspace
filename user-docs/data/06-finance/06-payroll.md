# Заробіток водіїв

{{SCRN:page_earnings_main}}

> Сторінка призначена для перегляду та аналізу заробітку водіїв, інтегрованих з Uklon.

## Огляд

Сторінка надає детальний фінансовий звіт щодо роботи водіїв за вибраний період. Вона підтримує два режими перегляду (стандартний та розширений), фільтрацію за датами та можливість експорту даних у форматі Excel.

## Реєстр даних

| Колонка | Опис | Логіка / Сортування |
| :--- | :--- | :--- |
| **Водій** | ПІБ водія | Клік відкриває {{MODAL:driver_earnings_details}} |
| **Кількість поїздок** | Загальна кількість виконаних поїздок | Сортування за кількістю |
| **Загальний дохід** | Сума грошових надходжень | Сортування за сумою |
| **Комісія Uklon** | Відрахована комісія | - |
| **Чистий заробіток** | Дохід мінус комісії та витрати | Сортування за значенням |

### Стани інтерфейсу

| Стан | Умова відображення | Скріншот |
| :--- | :--- | :--- |
| Мобільний | Ширина екрана < 768px | {{SCRN:page_earnings_mobile}} |

## Функціональні можливості

- **Вибір періоду:** Фільтрація звітів за довільним проміжком дат.
- **Розширений режим:** Відображення додаткових технічних та фінансових метрик.
- **Експорт:** Вивантаження звіту у форматі `.xlsx`.
- **Розрахункова інформація:** Перегляд методології нарахувань. Використовує {{MODAL:driver_earnings_calc_info}}.
- **Перегляд сирих даних:** Аналіз технічних даних по кожному водію. Використовує {{MODAL:driver_earnings_raw_data}}.

## Управління (Actions)

### Глобальні дії (хедер сторінки)

| Дія | Результат |
| :--- | :--- |
| **Експорт (Excel)** | Завантажує `.xlsx` файл зі звітом |
| **Розширений режим** | Перемикає відображення додаткових метрик |
| **Інформація** | Відкриває {{MODAL:driver_earnings_calc_info}} |

| :--- | :--- |

| **Сирі дані** | Відкриває {{MODAL:driver_earnings_raw_data}} |

## Технічні особливості

> Цей розділ призначений для розробників.

- **Локалізація:** Стан (діапазон дат, сортування, режим) зберігається в `localStorage`.
- **Експорт:** Реалізований через `apiClient` з отриманням `blob` і динамічним створенням посилання для завантаження.
- **Дані:** Робота з `dayjs` (включаючи `isoWeek`) для коректної обробки періодів.
