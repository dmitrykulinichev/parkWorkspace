# Заробіток автомобілів

{{SCRN:page_vehicle_earnings_main}}

> Сторінка надає детальний фінансовий звіт по кожному автомобілю, включаючи доходи від агрегаторів та прямих поїздок, витрати, бонуси та чистий прибуток.

## Огляд

Звіт дозволяє оцінити ефективність роботи автопарку. Дані агрегуються за вибраний період, надаючи інформацію про пробіг, кількість поїздок, дохідність водія та парку. Сторінка базується на даних фінансових транзакцій та звітів агрегаторів (Uklon).

## Реєстр даних

Таблиця відображає фінансові показники в розрізі автомобілів.

| Колонка | Опис | Логіка / Сортування |
| :--- | :--- | :--- |
| **Автомобіль** | Марка, модель та номер авто. | Сортування за назвою. |
| **Дохід (Gross)** | Загальний валовий дохід. | Сортування за значенням. |
| **Комісія Uklon** | Сума комісії, утриманої агрегатором. | Сортування за значенням. |
| **Витрати** | Сукупні витрати на авто. | Сортування за значенням. |
| **Прибуток (Net)** | Чистий прибуток після всіх відрахувань. | Сортування за значенням. |
| **Пробіг** | Загальна відстань, пройдена за період. | Сортування за значенням. |
| **Поїздки** | Загальна кількість виконаних замовлень. | Сортування за значенням. |

## Функціональні можливості

- **Фільтрація за датою:** Вибір періоду звітності через {{MODAL:date_range_filter}} (умовний ID).
- **Експорт даних:** Завантаження звіту у форматі `.xlsx` для подальшого аналізу.
- **Довідкова інформація:** Перегляд методології розрахунків через {{MODAL:vehicle_earnings_calc_info}}.
- **Деталізація:** Перегляд повної фінансової карти авто через {{MODAL:vehicle_earnings_details}}.

## Технічні особливості

- **Валідація:** Фільтрація здійснюється за допомогою `dayjs` для коректного вибору дат.
- **Синхронізація:** Дані автоматично оновлюються при зміні діапазону дат через `localStorage`.
- **Адаптивність:** Використовується окремий навігаційний блок для мобільних пристроїв.
- **Права доступу:** Доступ до сторінки обмежений дозволом `finance-vehicle-earnings`.
