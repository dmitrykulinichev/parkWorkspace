# Борги водіїв (Плани розстрочки)

{{SCRN:page_debt_plans_main}}

> Сторінка дозволяє керувати планами розстрочки (боргів) водіїв, відстежувати прогрес погашення та проводити списання коштів.

## Огляд

Сторінка забезпечує реєстр активних та завершених планів боргів водіїв. Вона відображає графік списань, прогрес погашення (у % та грошовому еквіваленті) та дозволяє адміністратору вручну списувати кошти згідно з планом або в рахунок погашення боргу.

## Реєстр даних

| Колонка | Опис | Логіка / Сортування |
| :--- | :--- | :--- |
| **Водій** | ПІБ водія | Клік відкриває {{MODAL:drv_driver_details}} |
| **Автомобіль** | Прив'язаний автомобіль | Клік відкриває {{MODAL:fleet_vehicle_details}} |
| **Загальна сума** | Сума боргу | Загальний борг |
| **Сплачено** | Сума погашення | Сплачена частина |
| **Прогрес** | Візуальний прогрес погашення | Progress bar (%), клік відкриває деталі |
| **Наступна сплата** | Дата та сума платежу | Дата, сума, частота |
| **Статус** | Активний / Призупинено / Завершено | Chip status |
| **Дії** | Кнопки управління | Перегляд, Редагувати, Списати, Пауза/Плей, Видалити |

## Функціональні можливості

- **Створення плану:** Створення розстрочки з вказанням загальної суми, суми платежу та частоти. Використовує {{MODAL:driver_debt_plan_form}}.
- **Редагування:** Зміна умов плану. Використовує {{MODAL:driver_debt_plan_form}}.
- **Ручне списання:** Погашення частини боргу. Використовує {{MODAL:manual_debt_payment_form}}.
- **Керування статусом:** Призупинення або відновлення плану.
- **Деталі:** Перегляд повної історії списань. Використовує {{MODAL:driver_debt_plan_details}}.

## Технічні особливості

> Цей розділ призначений для розробників.

- **Валідація:** Сума списання не може перевищувати залишок боргу (серверна перевірка та `validate` у модалці).
- **Синхронізація:** Використовується React Query для оновлення даних після транзакцій.
- **Статуси:** Логіка перемикання статусів (Active/Paused/Completed).
