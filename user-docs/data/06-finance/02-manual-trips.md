# Ручні поїздки

{{SCRN:page_manual_trips_main}}

> Сторінка призначена для ручного обліку поїздок, які не були зафіксовані автоматично.

## Огляд

Централізований реєстр ручних поїздок. Користувач може створювати записи про поїздки, прив'язуючи їх до автомобіля та водія.

## Функціональні можливості

- **Створення поїздки:** Відкриває {{MODAL:manual_trip_form}}.
- **Редагування:** Редагування існуючого запису через {{MODAL:manual_trip_form}}.
- **Перегляд деталей:** Деталі поїздки через {{MODAL:manual_trip_details}}.
- **Видалення:** Видалення запису.
