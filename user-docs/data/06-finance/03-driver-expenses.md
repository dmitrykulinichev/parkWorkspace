# Витрати водіїв

{{SCRN:page_drv_expenses_main}}

> Сторінка дозволяє переглядати, додавати, редагувати та видаляти фінансові витрати водіїв для подальшого врахування у розрахунках заробітної плати.

## Огляд

Сторінка є реєстром фінансових операцій, пов'язаних з витратами водіїв. Кожен запис прив'язаний до конкретного водія та автомобіля, має грошову суму та може містити долучені файли (чеки, квитанції). Модуль інтегрований з фінансовим обліком та впливає на підсумковий розрахунок балансу водія.

## Реєстр даних

| Колонка | Опис | Логіка / Сортування |
| :--- | :--- | :--- |
| **Дата** | Дата та час проведення витрати | Сортування за часом (новиші зверху) |
| **Автомобіль** | Автомобіль, до якого прив'язана витрата | Посилання на детальну картку авто |
| **Водій** | Водій, на якого нараховано витрату | Посилання на картку водія |
| **Сума** | Грошовий еквівалент витрати | Відображається в грн, виділена червоним |
| **Опис** | Текстовий коментар до витрати | Обмежено 30 символами з Tooltip |
| **Створив** | Користувач системи, що створив запис | Відображення аватара або імені |
| **Файли** | Наявність долучених документів | Кількість файлів з іконкою скріпки |
| **Дії** | Кнопки управління | Деталі, Редагувати, Видалити |

## Функціональні можливості

- **Додавання витрати:** Відкриває форму для створення нової витрати. Використовує {{MODAL:drv_exp_form}}.
- **Редагування:** Дозволяє змінити дані існуючої витрати. Використовує {{MODAL:drv_exp_form}}.
- **Видалення:** Видаляє витрату із системи (вимагає підтвердження).
- **Фільтрація:** Пошук за водієм, авто або датою. Використовує {{MODAL:drv_exp_filters}}.

## Технічні особливості

- **Валідація:** Сума має бути позитивною, дата не може бути в майбутньому.
- **Права доступу:** Доступно користувачам з дозволом `driver_expenses.view`.
- **Файли:** Дозволено завантаження зображень та PDF через `Paperclip` у формі.
