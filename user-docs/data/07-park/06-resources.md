# Матеріали для друку

{{SCRN:page_park_materials_main}}

> Сторінка призначена для генерації та друку інформаційних матеріалів парку: рекламних листівок, QR-кодів автомобілів та пам'яток для водіїв.

## Огляд

Сторінка «Матеріали для друку» дозволяє керівнику парку швидко отримати доступ до друкованих форм, необхідних для операційної діяльності та залучення водіїв. Вона автоматизує створення QR-кодів та пам'яток, забезпечуючи їх актуальність та відповідність брендингу парку.

## Функціональні можливості

- **Друк рекламної листівки:** Генерація PDF з QR-кодом на публічну сторінку лендінгу парку для рекрутингу.
- **Друк QR-кодів автомобілів:** Масова генерація наліпок з QR-кодами для всього автопарку.
- **Друк пам'ятки водія:** Генерація інструкції з правил роботи та контактів парку.

## Управління (Actions)

### Глобальні дії

| Дія | Результат |
| :--- | :--- |
| **Роздрукувати (Листівка)** | Відкриває сторінку друку рекламної листівки (А5) в новій вкладці. |
| **Роздрукувати всі (QR авто)** | Відкриває сторінку друку наліпок з QR-кодами для всіх активних авто. |
| **Роздрукувати (Пам'ятка)** | Відкриває сторінку друку пам'ятки водія в новій вкладці. |

## Технічні особливості

> Цей розділ призначений для розробників.

- **Формування запитів:** Усі дії друку підтримують режим демо (через `localStorage.getItem('demo_mode')`), який передається в URL параметрах.
- **Адаптивність:** Сторінка адаптована для мобільних пристроїв через `useIsMobile` та мобільне меню навігації.
- **Взаємодія:** Використовує вікно браузера (`window.open`) для відкриття сторінок друку.
