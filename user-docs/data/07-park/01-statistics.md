# Статистика парку

{{SCRN:page_park_stats_main}}

> Сторінка надає зведену аналітичну інформацію про діяльність парку, включаючи статистику замовлень Uklon та пробіг автомобілів.

## Огляд

Сторінка призначена для моніторингу ключових показників ефективності (KPI) роботи парку в сервісі Uklon. Вона об'єднує дані про замовлення та пробіг, дозволяючи адміністраторам парку швидко оцінювати поточний стан справ. Сторінка динамічно оновлюється та використовує візуалізацію даних для спрощення аналізу.

## Функціональні можливості

- **Перегляд статистики Uklon:** Відображення даних про замовлення (кількість, статус) через віджети `UklonOrdersStatsWidget` та `TestUklonOrdersStatsWidget`.
- **Аналіз пробігу:** Візуалізація даних пробігу автомобілів через графічний віджет `VehicleMileageChartWidget`.
- **Мобільна навігація:** Повна підтримка адаптивного меню для зручного доступу на мобільних пристроях.
