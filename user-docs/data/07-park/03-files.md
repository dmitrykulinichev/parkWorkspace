# Файли парку

{{SCRN:page_park_files_main}}

> Сторінка для зберігання та систематизації загальних документів парку, які не прив'язані до конкретних автомобілів чи водіїв.

## Огляд

Розділ "Файли парку" слугує централізованим сховищем для адміністративної документації: договорів оренди офісу, статутних документів, загальних страхових полісів тощо. Кожен запис може містити декілька прикріплених файлів, текстові примітки та фінансову складову (вартість), яка автоматично враховується в системі фінансів.

## Реєстр даних

| Колонка | Опис | Логіка / Сортування |
| :--- | :--- | :--- |
| **Файли** | Перелік завантажених файлів з посиланнями на перегляд. | Клік відкриває файл у новій вкладці. |
| **Примітки** | Текстовий опис або деталі документа. | Відображається зі збереженням форматування. |
| **Вартість** | Сума витрат, пов'язана з документом. | Відображається як сума або пов'язана транзакція. |
| **Створив** | Користувач, який додав документ у систему. | Посилання на картку користувача. |

## Функціональні можливості

- **Завантаження документів:** Пакетне завантаження файлів через {{MODAL:park_file_form}}.
- **Управління витратами:** Фіксація вартості документа, що автоматично створює фінансовий запис.
- **Швидкий пошук:** Текстовий пошук за примітками.
- **Фільтрація:** Відбір документів за періодом та автором через {{MODAL:park_file_filters}}.

## Управління (Actions)

### Глобальні дії (хедер сторінки)

| Дія | Результат |
| :--- | :--- |
| **Додати документ** | Відкриває форму {{MODAL:park_file_form}}. |
| **Пошук** | Пошук за вмістом приміток. |
| **Фільтри** | Відкриває модальне вікно {{MODAL:park_file_filters}}. |

### Дії з записом (рядкові)

| Дія | Результат |
| :--- | :--- |
| **Редагувати** | Відкриває форму {{MODAL:park_file_form}} для зміни приміток чи додавання файлів. |
| **Видалити** | Відкриває запит на підтвердження {{MODAL:park_file_delete}}. |

## Технічні особливості

- **FileManager:** Використовує спільний компонент для управління файлами (завантаження, видалення, прев'ю).
- **Фінансова інтеграція:** Поле `cost` при збереженні документа може тригерувати створення фінансової транзакції в системі.
- **Права доступу:** Доступ до розділу визначається загальним дозволом `park`.
