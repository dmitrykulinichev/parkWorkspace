# Налаштування фінансів

{{SCRN:page_park_finance_settings_main}}

> Централізоване управління фінансовими параметрами парку: відсотки комісій, норми доходу та KPI для автомобілів.

## Огляд

Сторінка дозволяє редагувати поточні фінансові налаштування (такі як комісія Uklon, база відсотка водія тощо). Збереження будь-яких змін автоматично фіксується в системі та створює новий запис в історії, що вступає в дію з поточної дати.

## Функціональні можливості

- **Комісія Uklon (%):** Налаштування відсотка комісії агрегатора.
- **Базовий відсоток водія (%):** Встановлення частки чистого прибутку, що йде водієві.
- **Мінімальний дохід парку (грн/день):** Фінансовий орієнтир для денної виручки парку.
- **Норма поїздок на місяць:** KPI для автомобілів (кількість поїздок).
- **Історія змін:** Перегляд попередніх версій фінансових налаштувань.

## Управління (Actions)

### Глобальні дії

| Дія | Результат |
| :--- | :--- |
| **Зберегти налаштування** | Оновлює поточні параметри та створює новий запис в історії. |

## Технічна інформація

| Файл | Роль |
| :--- | :--- |
| `resources/js/pages/FinanceSettings/index.jsx` | Головний компонент сторінки налаштувань. |
| `resources/js/hooks/useFinanceSettings.js` | Хук логіки та запитів (React Query). |
| `resources/js/components/FinanceSettings/FinanceHistoryTable.jsx` | Компонент таблиці історії змін. |

## Технічні особливості

- **Валідація:** Поля приймають числові значення з обмеженнями (відсотки до 100, додатні суми).
- **Синхронізація:** Зміна налаштувань є системною подією, яка автоматично оновлює бізнес-логіку розрахунків для всіх нових транзакцій від моменту збереження.
