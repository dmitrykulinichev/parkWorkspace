# Налаштування Лендінгу

{{SCRN:page_park_landing_main}}

> Сторінка дозволяє керувати контентом публічної сторінки парку (лендінгу), включаючи візуальне оформлення, контакти та стрічку новин.

## Огляд

Сторінка розділена на вкладки для управління різними аспектами лендінгу. Вона дозволяє налаштовувати SEO-параметри (slug), зовнішній вигляд (лого, кольори, херо-зображення), інформаційні блоки (переваги, вимоги, FAQ) та публікувати новини для водіїв.

## Вкладки

### Загальні налаштування

{{SCRN:tab_park_landing_general}}

> Основне налаштування публічного профілю парку та його візуального оформлення.

#### Функціональні можливості

- **Публічне посилання:** Встановлення унікального слага (`/p/{slug}`) та перемикач активності лендінгу.
- **Основна інформація:** Редагування заголовків, описів, завантаження логотипа та херо-зображення.
- **Візуалізація:** Налаштування брендових кольорів та прозорості оверлею херо-блоку.
- **Інформаційні блоки:** Управління списком переваг (benefits), вимог до водіїв та розділом FAQ.
- **Контакти:** Встановлення контактного телефону та посилання на Telegram.

#### Управління (Actions)

| Дія | Результат |
| :--- | :--- |
| **Копіювати** | Копіює повне посилання на лендінг у буфер обміну. |
| **Відкрити** | Відкриває лендінг у новій вкладці (якщо активний). |
| **Зберегти** | Зберігає всі внесені налаштування лендінгу. |

### Новини та акції

{{SCRN:tab_park_landing_news}}

> Реєстр публікацій, що відображаються у блоці новин на лендінгу.

#### Реєстр даних

| Колонка | Опис | Логіка / Сортування |
| :--- | :--- | :--- |
| **Статус** | Перемикач відображення новини на лендінгу | Вкл/Викл |
| **Заголовок / Slug** | Назва новини та її URL-фрагмент | Пошук за назвою |
| **Дата публікації** | Дата створення або публікації | Сортування за датою |

#### Функціональні можливості

- **Додавання новини:** Використовує {{MODAL:landing_news_form}}.
- **Редагування:** Дозволяє змінювати контент та обкладинку новини. Використовує {{MODAL:landing_news_form}}.
- **Видалення:** Використовує {{MODAL:landing_news_delete}}.

#### Управління (Actions)

| Дія | Результат |
| :--- | :--- |
| **Додати новину** | Відкриває {{MODAL:landing_news_form}} для створення нової публікації. |
| **Редагувати** | Відкриває {{MODAL:landing_news_form}} з даними новини. |
| **Видалити** | Відкриває {{MODAL:landing_news_delete}} для підтвердження. |

## Технічна інформація

> Цей розділ призначений для розробників.

| Файл | Роль |
| :--- | :--- |
| `resources/js/pages/LandingSettings/index.jsx` | Головний компонент сторінки, управління табами |
| `resources/js/pages/LandingSettings/tabs/GeneralSettingsTab.jsx` | Логіка та UI загальних налаштувань |
| `resources/js/pages/LandingSettings/tabs/NewsAndPromotionsTab.jsx` | Логіка та UI стрічки новин |
| `resources/js/hooks/useSettings.js` | Сервіс API для запитів до лендінгу та новин |

## Технічні особливості

> Цей розділ призначений для розробників.

- **Синхронізація:** Стан лендінгу синхронізується при завантаженні через `loadLandingSettings`.
- **Файли:** Завантаження логотипа та херо-зображення виконується через сервіс `useFileUpload`.
- **Slugify:** Новини автоматично отримують slug на основі заголовка при створенні.
- **Стани:** Мобільна адаптація підтримується окремим компонентом меню `MobileNavigationMenu`.
