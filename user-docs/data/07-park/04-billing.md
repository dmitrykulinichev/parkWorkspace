# Білінг та Тарифи

{{SCRN:page_park_billing_main}}

> Сторінка дозволяє переглядати поточний статус підписки, розрахунок суми оплати за поточний період та історію попередніх нарахувань.

## Огляд

Сторінка відображає дані про підписку парку. Вона показує активний розрахунковий період, розбивку суми оплати (базова абонплата + плата за активні автомобілі) та таблицю з усіма минулими періодами білінгу.

## Реєстр даних

| Колонка | Опис | Логіка / Сортування |
| :--- | :--- | :--- |
| **Період** | Дати початку та закінчення розрахункового періоду | Сортування за датою |
| **Тариф** | Назва підписки | - |
| **К-сть авто** | Кількість активних автомобілів у парку за цей період | - |
| **Сума** | Загальна сума до оплати за період | - |
| **Статус** | Поточний статус (Активний, Завершено, Оплачено) | - |

## Технічні особливості

> Цей розділ призначений для розробників.

- **Синхронізація:** Дані завантажуються через React Query з ключем `['billingHistory', user?.park?.id]`.
- **API-запит:** GET `/billing`.
- **Валідація:** Дані формуються серверною частиною; на клієнті виконується лише форматування дат через `dayjs`.
