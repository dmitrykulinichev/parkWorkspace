# Модальне вікно: Передача автомобіля (Трансфер)

{{MODAL:modal_fleet_vehicle_transfer}}

Спеціалізований інструмент для переміщення активів між різними парками всередині платформи (мульти-аккаунт).

## Функціонал вікна

### 1. Вибір цільового парку
Доступні лише ті парки, до яких поточний користувач має доступ і які відрізняються від поточного.

### 2. Опції копіювання даних
При передачі ви можете обрати, які саме історичні дані "поїдуть" разом з авто:
- **Документи:** Скан-копії техпаспортів тощо.
- **Страховки:** Активні поліси ОСЦПВ/КАСКО.
- **Регламентні роботи:** Налаштовані графіки ТО.
- **Історія ремонтів:** Повний архів заїздів на сервіс.

### 3. Примітка
Текстове поле для пояснення причини трансферу.

## Попередження (Alert)
Після передачі:
1.  Автомобіль стає неактивним у поточному парку.
2.  Всі активні призначення водіїв автоматично завершуються.

## Технічна інформація

| Файл | Роль |
| :--- | :--- |
| `resources/js/components/modals/VehicleTransferModal.jsx` | Компонент вікна передачі. |
