# Модальне вікно: Форма новини

{{SCRN:modal_landing_news_form}}

> Вікно для створення нової публікації або редагування існуючої новини на лендінгу.

## Поля форми

| Поле | Тип | Обов'язкове | Опис / Технічні деталі |
| :--- | :--- | :---: | :--- |
| **Заголовок** | text | Так | Використовується для формування автоматичного слага. |
| **Slug** | text | Так | Унікальний URL-ідентифікатор новини. |
| **Зміст** | textarea | Так | Основний текст новини (підтримується plain text). |
| **Зображення** | file | Ні | Завантаження обкладинки (JPG, PNG, WebP). |
| **Активна** | toggle | Так | Визначає, чи буде новина видимою на сайті. |

## Логіка та валідація

- **Авто-slug:** При введенні заголовка slug генерується автоматично (транслітерація кирилиці).
- **Зображення:** Підтримується прев'ю, видалення існуючого зображення та завантаження нового (до 5 МБ).
- **Обробка помилок:** Серверна перевірка унікальності слага.
- **Після збереження:** Список новин оновлюється без перезавантаження сторінки.
