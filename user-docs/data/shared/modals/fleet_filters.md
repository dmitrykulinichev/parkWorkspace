# Модальне вікно: Фільтрація автомобілів

{{MODAL:modal_fleet_filters}}

Це вікно дозволяє виконувати точне налаштування відображення списку автомобілів за допомогою набору параметрів.

## Доступні фільтри

| Параметр | Опис | Тип вводу |
| :--- | :--- | :--- |
| **Статус** | Фільтрація за поточним операційним станом авто. | Select (Вільний, В роботі тощо) |
| **Марка** | Вибір конкретного виробника. | Select (динамічний список) |
| **Модель** | Вибір моделі автомобіля. | Select (залежить від марки) |
| **Водій** | Пошук автомобілів, закріплених за конкретним водієм. | Select (пошук по ПІБ) |
| **Рік** | Фільтрація за роком виробництва. | Select |
| **Водій (Наявність)** | Швидка фільтрація за фактом наявності будь-якого водія. | Select (З водієм / Без водія) |

## Логіка роботи
- **Множинність:** Ви можете комбінувати декілька фільтрів одночасно (наприклад, "Toyota" + "Вільний" + "2022 рік").
- **Скидання:** Кнопка "Скинути всі" дозволяє миттєво повернути таблицю до початкового стану.
- **Візуалізація:** Обрані фільтри відображаються у вигляді "чіпів" (badges) над таблицею, що дозволяє швидко видалити конкретний критерій без відкриття модального вікна.
- **Динамічність:** Список моделей автоматично оновлюється при зміні обраної марки.

## Технічна інформація

| Файл | Роль |
| :--- | :--- |
| `resources/js/components/UI/FiltersModal.jsx` | Універсальний компонент для відображення фільтрів. |
| `resources/js/hooks/vehicle/useVehiclesPage.js` | Формує масив `filterConfig` з опціями та типами фільтрів. |
| `resources/js/hooks/useUrlFilters.js` | Логіка синхронізації станів фільтрів з URL-параметрами (query strings). |
| `resources/js/services/filterOptionsService.js` | (Частково) Завантаження доступних значень для селектів. |
