# Модальне вікно: Прийняти рішення по звіту

{{SCRN:modal_issue_report_actions}}

> Вікно для вибору рішення за зверненням: створення задачі, відхилення або закриття звіту.

## Логіка та валідація

- **Логіка:** Доступне лише для звітів зі статусами `new` та `needs_info`.
- **Результат:** Може створити сутність `Task` у системі або змінити статус звіту на `resolved` / `rejected`.
