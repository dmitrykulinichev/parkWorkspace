# Модальне вікно: Деталі витрати водія

{{SCRN:modal_drv_exp_details}}

> Вікно для перегляду повної інформації про фінансову витрату водія, включаючи метадані та прикріплені документи.

## Огляд

Вікно відображає таблицю з детальними параметрами витрати: дату, прив'язаний автомобіль та водія, суму, опис та інформацію про автора запису. Також містить список усіх прикріплених файлів з можливістю їх перегляду/завантаження.

## Логіка перегляду

- **Автоматичне завантаження:** При відкритті вікна виконується запит на отримання повних даних витрати, включаючи список файлів.
- **Файли:** Відображається кількість файлів. Якщо файли є, формується їх список з назвами, розміром та іконкою завантаження.
- **Закриття:** Вікно закривається кнопкою "Закрити".
