# Модальне вікно: Підключення Telegram

{{SCRN:modal_drv_telegram}}

> Вікно керування підключенням водія до Telegram-бота. Дозволяє генерувати код підключення та відключати бота. Відкривається кліком на Telegram-значок у таблиці або через пункт "Telegram" у контекстному меню сторінки `/drivers`.

## Логіка та валідація

- **Режими вікна:** Визначаються поточним статусом підключення:
  - `not_connected` — кнопка "Підключити" для генерації нового коду.
  - `pending` — відображається QR-код та посилання для підключення. Код дійсний 24 години.
  - `connected` — підтвердження успішного підключення та кнопка "Відключити".
- **Після генерації коду:** Оновлюється статус водія у кеші.
- **Після відключення:** Telegram ID та токен очищуються на сервері. Оновлюється статус водія.
