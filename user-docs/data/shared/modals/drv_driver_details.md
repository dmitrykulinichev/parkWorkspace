# Модальне вікно: Деталі водія

{{SCRN:modal_drv_driver_details}}

> Картка перегляду повного профілю водія. Відкривається кліком на рядок таблиці або через пункт "Показати деталі" в контекстному меню на сторінці `/drivers`.

## Логіка та валідація

- **Режими вікна:** Лише перегляд. Редагування та видалення доступні через кнопки у вікні.
- **Після збереження:** Зміна статусу активності через кнопку-перемикач всередині вікна — виклик мутації `toggleDriverStatus`. При успіху інвалідується кеш водія і списку.
