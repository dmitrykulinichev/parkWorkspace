# Модальне вікно: Повідомити про проблему

{{MODAL:modal_maint_issue_report_form}}

Форма для швидкої реєстрації технічних несправностей або інших інцидентів, пов'язаних з автомобілем.

## Поля форми

| Поле | Опис | Технічні деталі |
| :--- | :--- | :--- |
| **Автомобіль** | Вибір ТЗ, на якому виявлено проблему. | Обов'язкове. |
| **Опис звіту** | Детальний опис несправності. | Обов'язкове текстове поле. |
| **Повідомив водій** | Хто саме ініціював звернення. | Вибір зі списку водіїв. |
| **Дата повідомлення**| Коли було виявлено проблему. | За замовчуванням — поточний час. |

## Логіка роботи
- **Автозаповнення:** Якщо модалка викликається з рядка конкретного авто, поле "Автомобіль" буде заповнене автоматично.
- **Подальші кроки:** Створений звіт з'являється в розділі "Звернення" і може бути конвертований менеджером у запис ТО або Ремонт.

## Технічна інформація

| Файл | Роль |
| :--- | :--- |
| `resources/js/components/modals/issueReport/IssueReportFormModal.jsx` | Компонент форми повідомлення. |
| `resources/js/queries/issueReports.js` | React Query хук `useIssueReportModal` та API-мутація. |
| `resources/js/utils/dateUtils.js` | Утиліти для роботи з часом повідомлення. |
