# Модальне вікно: Фільтри команди

{{SCRN:modal_park_user_filters}}

> Призначене для фільтрації списку користувачів за заданими критеріями.

## Поля форми

| Поле | Тип | Обов'язкове | Опис / Технічні деталі |
| :--- | :--- | :---: | :--- |
| **Роль** | select | Ні | Фільтрація за системною роллю. |
| **Статус** | select | Ні | Фільтрація за статусом активації (Активний/Очікує активації). |
| **Telegram** | select | Ні | Фільтрація за наявністю підключеного Telegram-аккаунта. |

## Логіка та валідація

- **Скидання фільтрів:** Кнопка "Очистити" повертає список до початкового стану без фільтрів.
- **Застосування:** Фільтри застосовуються миттєво при натисканні кнопки "Показати результати".
