# Модальне вікно: Форма задачі

{{SCRN:modal_task_form}}

> Вікно для створення нової задачі або редагування існуючої. Дозволяє призначити задачу на конкретний автомобіль та виконавця.

## Поля форми

| Поле | Тип | Обов'язкове | Опис / Технічні деталі |
| :--- | :--- | :---: | :--- |
| **Заголовок** | text | Так | Коротка назва задачі. |
| **Опис** | textarea | Ні | Детальний опис робіт або проблеми. Підтримує багаторядкове введення. |
| **Автомобіль** | vehicle_select | Ні | Прив'язка задачі до транспортного засобу. Відображає список авто парку з їх статусами. |
| **Пріоритет** | select | Так | Важливість задачі: Низький, Середній, Високий, Критичний. |
| **Виконавець** | user_select | Ні | Користувач системи, відповідальний за виконання. Відображає список команди парку. |

## Логіка та валідація

- **Обов'язкові поля:** Заголовок та Пріоритет.
- **Режими вікна:**
    - **Створення:** Поля порожні або мають значення за замовчуванням (Середній пріоритет).
    - **Редагування:** Заповнюється даними існуючої задачі.
- **Автофокус:** При відкритті фокус встановлюється на перше поле введення (Заголовок).
- **Валідація:** При спробі збереження без обов'язкових полів відображаються підказки, а екран прокручується до першої помилки.
