# Модальне вікно: Створення/редагування витрати водія

{{SCRN:modal_drv_exp_form}}

> Форма для внесення нової або редагування існуючої фінансової витрати водія.

## Поля форми

| Поле | Тип | Обов'язкове | Опис / Технічні деталі |
| :--- | :--- | :---: | :--- |
| **Автомобіль** | select | Так | Обирається зі списку автопарку |
| **Водій** | select | Ні | Водій, на якого проводиться нарахування |
| **Сума** | number | Так | Грошова сума витрати |
| **Дата та час** | datetime | Так | Дата і час фіксації витрати (за замовчуванням - поточний час) |
| **Опис витрати** | textarea | Ні | Коментар або деталі витрати |
| **Файли** | file | Ні | Завантаження чеків або підтверджуючих документів (до 5 файлів) |

## Логіка та валідація

- **Обов'язкові поля:** Автомобіль, сума, дата та час.
- **Після збереження:** Створений або оновлений запис відображається у реєстрі витрат водіїв.
- **Завантаження файлів:** Реалізовано через `FileManager`, підтримується завантаження кількох файлів одночасно.
