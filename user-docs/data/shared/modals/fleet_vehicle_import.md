# Модальне вікно: Імпорт автомобілів

{{MODAL:modal_fleet_vehicle_import}}

Інструмент для пакетного додавання великої кількості автомобілів через файли Excel (.xlsx, .xls) або CSV.

## Вкладки

### 1. Новий імпорт
Процес складається з двох кроків:

#### Крок 1: Завантаження (Upload)
- **Область завантаження:** Підтримка Drag-and-drop або вибір файлу через провідник.
- **Шаблони:** Можливість завантажити еталонні файли у форматах `.xlsx`, `.xls` або `.csv` для правильного заповнення даних.

#### Крок 2: Попередній перегляд (Preview)
- **Статистика:** Зведений звіт (Всього рядків, Валідні, Дублікати, Помилки).
- **Інтерактивна таблиця:** Відображення даних з файлу перед завантаженням у базу.
- **Кольорове маркування:**
    - **Білий:** Валідні дані.
    - **Помаранчевий:** Виявлено дублікат (по VIN або номеру) — рядок буде пропущено.
    - **Червоний:** Помилка валідації (невірний формат року, відсутнє обов'язкове поле тощо) — рядок буде пропущено.

### 2. Історія імпортів
Журнал усіх проведених завантажень:
- Дата та назва файлу.
- Статус (Успішно, Частково, Помилка).
- Результат (кількість імпортованих записів проти загальної кількості).
- Користувач, який ініціював імпорт.
- **Деталі:** При кліку на запис відкривається детальний лог з результатами по кожному рядку.

## Логіка роботи
1.  **Валідація:** Система автоматично перевіряє VIN та державний номер на унікальність перед імпортом.
2.  **Розумний парсинг:** Підтримка різних назв колонок (напр., "Марка", "Make", "Brand").
3.  **Безпека:** Записи з помилками або дублікатами автоматично відфільтровуються, дозволяючи імпортувати тільки чисті дані.
4.  **Аудит:** Всі імпортовані автомобілі отримують мітку про джерело походження (Batch ID).
