# Модальне вікно: Деталі автомобіля

{{MODAL:modal_fleet_vehicle_details}}

Це вікно надає повний огляд інформації про конкретний транспортний засіб, згрупований за категоріями.

## Структура вікна

### 1. Header (Заголовок)
- **Фото:** Відображається завантажене фото або іконка за замовчуванням.
- **Назва:** Марка та модель автомобіля.
- **Рік випуску:** Рік виробництва ТЗ.
- **Статус:** Кольоровий бейдж поточного стану (В роботі, Вільний тощо).

### 2. Основна інформація (Блок 1)
- **Номерний знак:** Реєстраційний номер (Bold, Monospace).
- **VIN код:** Ідентифікаційний номер кузова (Monospace).
- **Марка / Модель:** Текстові назви.
- **Пробіг:** Останній зафіксований показник у км.
- **Паливо:** Список типів пального, що використовуються (у вигляді чіпів).

### 3. Водій (Блок 2)
- Відображається лише за наявності активного призначення.
- Містить ім'я водія та його номер телефону для швидкого зв'язку.

### 4. Характеристики та обладнання (Блок 3)
- **Технічні дані:** Тип кузова, Колір, Рівень комфорту, Кількість місць.
- **Додаткове обладнання:** Список опцій (Кондиціонер, Шкіряний салон тощо) у вигляді чіпів.

### 5. Системна інформація (Footer)
- Дата створення запису.
- Дата останнього оновлення.
- Дата додавання в автопарк.
- Розрахунковий вік автомобіля в роках.

## Доступні дії з вікна
- **Редагувати:** Перехід до форми редагування {{MODAL:fleet_vehicle_form}}.
- **Оновити пробіг:** Перехід до форми {{MODAL:fleet_mileage_update}}.
- **Видалити:** Викликає підтвердження видалення {{MODAL:fleet_vehicle_delete}}.
- **Закрити:** Повернення до загального списку.
