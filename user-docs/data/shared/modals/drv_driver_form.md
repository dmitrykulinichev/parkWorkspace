# Модальне вікно: Додати / Редагувати водія

{{SCRN:modal_drv_driver_form}}

> Форма створення або редагування профілю водія. Відкривається з кнопки "Додати водія" або з пункту "Редагувати" в контекстному меню рядка на сторінці `/drivers`.

## Поля форми

| Поле | Тип | Обов'язкове | Опис / Технічні деталі |
| :--- | :--- | :---: | :--- |
| **Прізвище** | text | Так | Прізвище водія |
| **Ім'я** | text | Так | Ім'я водія |
| **По-батькові** | text | Ні | По-батькові водія |
| **Дата народження** | date | Ні | Вибір з DatePicker-компонента |
| **Статус** | select | Так | Робочий статус. Значення отримуються з бекенду. За замовчуванням — `new` (Новий) |
| **Телефон** | text | Так | Номер телефону у форматі +380... |
| **Email** | email | Ні | Електронна пошта |
| **Номер посвідчення** | text | Ні | Номер водійського посвідчення |
| **Адреса реєстрації** | text | Ні | Офіційна адреса реєстрації |
| **Адреса проживання** | text | Ні | Фактична адреса проживання |
| **Телефон родича 1** | text | Ні | Контактний телефон першого родича (+380...) |
| **Примітка до родича 1** | text | Ні | Хто це: дружина, брат тощо |
| **Телефон родича 2** | text | Ні | Контактний телефон другого родича |
| **Примітка до родича 2** | text | Ні | Хто це: батько, сестра тощо |
| **Коментар** | textarea | Ні | Довільна додаткова інформація про водія |
| **Фото профілю** | file (image) | Ні | Доступно лише в режимі редагування. Upload зображення будь-якого формату |

## Логіка та валідація

- **Обов'язкові поля:** Прізвище, Ім'я, Телефон, Статус.
- **Режими вікна:** Режим "Додати" — аватар недоступний (завантаження фото можливе лише після створення запису). Режим "Редагувати" — доступний блок зміни/видалення аватара.
- **Обробка помилок:** Помилки валідації повертаються сервером у форматі `{field: [messages]}` і відображаються під відповідними полями. При помилці сторінка прокручується до першого проблемного поля.
- **Після збереження:** Інвалідується кеш `driversKeys.lists()`. У режимі редагування — також `driversKeys.detail(id)`. Відображається toast-повідомлення.
