# Модальне вікно: Деталі активності

{{SCRN:modal_activity_log_details}}

> Картка з повною інформацією про подію: користувач, час, технічні дані, тип дії та змінені поля.

## Поля (інформація)

| Блок | Параметр | Опис |
| :--- | :--- | :--- |
| **Користувач** | Ім'я / Email | Ініціатор події (або "Система"). |
| **Інформація** | Час дії | Дата та час події. |
| | IP адреса | IP-адреса, з якої виконано дію. |
| | Локація | Географічна локація (якщо доступно). |
| | Тривалість сесії | Час активності користувача. |
| **Технічна** | Пристрій | Тип пристрою та його тип (веб/мобільний). |
| | Браузер / ОС | Технічні дані клієнта. |
| | User Agent | Повний рядок браузера. |
| **Спеціальні** | Причина виходу | Додаткова інформація (якщо доступна). |
| | Змінені поля | Список полів, що були змінені (залежить від логіки). |
| | Безпека | Попередження, якщо подія критична. |

## Логіка та валідація

- **Відкриття:** Відкривається при кліку на запис у таблиці журналу подій.
- **Типи подій:** Візуально виділяються (статус "критична" змінює колір іконок та додає попередження безпеки).
- **Змінені поля:** Відображаються у вигляді чипів, якщо метадані події містять інформацію про зміни.
