# Модальне вікно: Швидка картка водія

{{SCRN:modal_drv_simple_driver_details}}

> Вікно призначене для миттєвого перегляду ключової інформації про водія без переходу в основний реєстр водіїв. Відкривається при кліку на ПІБ водія в графіку або пулі водіїв.

## Опис інтерфейсу

### Основна інформація
- **Аватар:** Фотографія водія.
- **ПІБ та статуси:** Повне ім'я та кольорові бейджі (Активний, Новий, Неактивний).
- **Telegram:** Індикатор підключеного Telegram-бота.

### Контакти
- **Телефон:** Форматований номер телефону.
- **Email:** Адреса електронної пошти.
- **Адреса проживання:** Актуальне місце проживання.

### Документи та Особисті дані
- **Посвідчення водія:** Номер документа та відкриті категорії (наприклад, A, B).
- **Дата народження:** Дата та автоматично розрахований повний вік (років).

### Поточне авто
Якщо водій має активне призначення, відображається картка поточного автомобіля (бренд, модель, номер).

## Логіка та валідація

- **Завантаження:** Дані завантажуються за запитом за ID водія. Використовується кешування `react-query`, що забезпечує миттєве відкриття при повторних кліках.
- **Пусті поля:** Якщо певна інформація (наприклад, адреса або номер ліцензії) не заповнена в профілі водія, відповідний блок автоматично приховується.
