# Модальне вікно: Додати/Редагувати автомобіль

{{MODAL:modal_fleet_vehicle_form}}

Це вікно призначене для внесення повної інформації про транспортний засіб у систему. Воно розділене на логічні блоки для зручності заповнення.

## Основні параметри (Секція "Основні дані")

| Поле | Опис | Технічні деталі |
| :--- | :--- | :--- |
| **Фото автомобіля** | Візуальне зображення ТЗ. | Доступно тільки в режимі редагування. Впливає на відображення на публічній сторінці. |
| **Теги** | Кольорові мітки для класифікації. | Можна вибрати декілька тегів зі словника. |
| **Пріоритетне авто** | Перемикач (Toggle) особливого статусу. | Використовується для виділення важливих авто в списках. |
| **Марка** | Виробник автомобіля (напр., Toyota). | Autocomplete з можливістю введення власного значення (FreeSolo). |
| **Модель** | Конкретна модель (напр., Camry). | Залежить від обраної марки. Autocomplete з можливістю введення власного значення. |
| **Рік** | Рік випуску автомобіля. | Вибір зі списку доступних років. |
| **Номерний знак** | Державний реєстраційний номер. | Формат: AA1234BB. Обов'язкове поле. |
| **VIN код** | Унікальний ідентифікатор кузова. | 17 символів. Перевіряється на унікальність у системі. |
| **Пробіг (км)** | Поточні показники одометра. | Обов'язкове при створенні. При редагуванні поле заблоковане (використовуйте окрему форму оновлення пробігу). |
| **Статус** | Початковий стан автомобіля. | Вільний, В роботі, Ремонт, Недоступний. При редагуванні змінюється системно. |
| **Тип палива** | Вид пального (Бензин, Газ, Електро тощо). | **Мультиселект**: можна вказати декілька (напр., Бензин + Газ). |

## Характеристики (Секція "Характеристики")

| Поле | Опис | Технічні деталі |
| :--- | :--- | :--- |
| **Кузов** | Тип кузова (Седан, Універсал тощо). | Вибір зі списку (bodyTypes). |
| **Колір** | Колір автомобіля. | Вибір зі списку системних кольорів. |
| **Комфорт** | Рівень комфорту (напр., Економ, Комфорт, Бізнес). | Використовується для фільтрації в інтеграціях. |
| **Місць** | Кількість пасажирських місць. | Числове поле (NumberStepper) від 1 до 50. |
| **Додаткові опції** | Оснащення авто (Кондиціонер, Шкіряний салон тощо). | **Мультиселект**: вибір зі списку доступних опцій. |

## Логіка та валідація
- **Залежність полів:** Список моделей фільтрується автоматично після вибору марки. Поле "Модель" заблоковане до моменту вибору марки.
- **Обов'язкові поля:** Позначені зірочкою `*`. Система не дозволить зберегти форму без їх заповнення.
- **Блокування при редагуванні:** Поля "Пробіг" та "Статус" заблоковані при редагуванні існуючого авто для забезпечення цілісності історії змін.
