# Модальне вікно: Фільтри документів

{{MODAL:modal_fleet_document_filters}}

Інструмент для швидкого сегментування реєстру документів за технічними та статусними ознаками.

## Доступні фільтри

| Параметр | Опис |
| :--- | :--- |
| **Тип документа** | Вибір конкретної категорії (напр. лише ОСЦПВ). |
| **Статус** | Фільтрація за станом: Активний, Прострочений, Закінчується. |
| **Автомобіль** | Відображення документів лише для обраної одиниці техніки. |

## Технічна інформація

| Файл | Роль |
| :--- | :--- |
| `resources/js/components/UI/FiltersModal.jsx` | Презентаційний компонент. |
| `resources/js/hooks/useDocumentsPage.js` | Конфігурація `filterConfig` з опціями типів та статусів. |
| `resources/js/services/filterOptionsService.js` | Завантаження динамічного списку автомобілів для селектора. |
