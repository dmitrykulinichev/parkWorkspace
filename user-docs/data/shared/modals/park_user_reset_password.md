# Модальне вікно: Скидання пароля

{{SCRN:modal_park_user_reset_password}}

> Підтвердження відправки запиту на встановлення нового пароля.

## Логіка та валідація

- **Для активних користувачів:** Надсилає стандартний лист для скидання пароля.
- **Для неактивних користувачів:** Повторно надсилає лист-запрошення (Activation Link).
- **Після підтвердження:** Система повідомляє про успішну відправку листа.
