# Модальне вікно: Фільтри журналу подій

{{SCRN:modal_activity_log_filters}}

> Вікно для детальної фільтрації подій у журналі за різними критеріями.

## Поля форми

| Поле | Тип | Опис / Технічні деталі |
| :--- | :--- | :--- |
| **Користувач** | select | Фільтрація подій за конкретним користувачем, який ініціював дію. |
| **Тип дії** | select (multiple) | Тип операції (наприклад: `login`, `vehicle_create`, тощо). |
| **Дата від** | date | Початок періоду подій. |
| **Дата до** | date | Кінець періоду подій. |
| **Критичні події** | checkbox | Фільтр для відображення подій з підвищеним рівнем безпеки. |

## Логіка та валідація

- **Скидання:** Кнопка "Скинути всі фільтри" видаляє всі вибрані критерії та закриває вікно.
- **Стан:** Вибрані фільтри відображаються чіпами на сторінці після застосування.

## Технічна інформація

| Файл | Роль |
| :--- | :--- |
| `resources/js/components/UI/FiltersModal.jsx` | Компонент загального вікна фільтрів. |
| `resources/js/hooks/useActivityLogsPage.js` | Логіка обробки стану фільтрів для журналу подій. |
