# Модальне вікно: QR-код автомобіля

{{MODAL:modal_fleet_vehicle_qr}}

Вікно для генерації та перегляду унікального ідентифікатора автомобіля, який забезпечує швидкий доступ до його публічної сторінки.

## Компоненти вікна

### 1. Візуальний QR-код
- Динамічно згенероване зображення, яке веде на публічний URL автомобіля.
- Оптимізоване для сканування смартфонами.

### 2. Публічне посилання
- Текстове поле (Read-only) з повним URL.
- **Кнопка Копіювати:** Дозволяє миттєво скопіювати лінк у буфер обміну.
- **Кнопка Відкрити:** Відкриває публічну сторінку автомобіля в новій вкладці браузера.

### 3. Друк (Print)
- Кнопка "Друкувати" відкриває спеціальну версію сторінки, підготовлену для принтера (наприклад, для створення наклейки на лобове скло або приладову панель).

## Призначення
QR-код дозволяє водіям або персоналу парку отримувати швидкий доступ до технічної інформації або форми повідомлення про проблему без необхідності авторизації в основній системі.

## Технічна інформація

| Файл | Роль |
| :--- | :--- |
| `resources/js/components/modals/VehicleQrModal.jsx` | Компонент відображення коду та посилань. |
| `API: /api/v1/vehicles/{id}/qr-code` | Ендпоінт генерації зображення QR-коду. |
| `public_access_token` | Поле моделі Vehicle, на основі якого будується публічний URL. |
| `resources/js/context/ToastContext.jsx` | Повідомлення "Посилання скопійовано". |
