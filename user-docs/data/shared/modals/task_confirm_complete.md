# Модальне вікно: Підтвердження завершення задачі

{{SCRN:modal_task_confirm_complete}}

> Вікно для підтвердження успішного виконання задачі.

## Логіка та валідація

- **Призначення:** Запобігає випадковому переведенню задачі у статус "Завершена".
- **Дія:** При підтвердженні задача отримує статус `completed`, фіксується час завершення та користувач, який виконав дію.
- **Результат:** Задача переміщується у відповідний розділ (або приховується, залежно від фільтрів) та стає доступною для перегляду в архіві.
