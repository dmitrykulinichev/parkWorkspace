# Модальне вікно: Видалення автомобіля

{{MODAL:modal_fleet_vehicle_delete}}

Вікно підтвердження безповоротної операції видалення транспортного засобу з бази даних парку.

## Вміст вікна
- **Заголовок:** "Видалити автомобіль".
- **Повідомлення:** Уточнюючий запит, що містить Марку, Модель та Державний номер автомобіля для запобігання помилковому видаленню.
- **Кнопка Видалити:** Виконана у небезпечному (червоному) стилі. При натисканні ініціює запит до API.
- **Кнопка Скасувати:** Закриває вікно без внесення змін.

## Технічні аспекти
- **Безпека:** Операція видалення зазвичай доступна лише користувачам з правами `Owner` або `Admin`.
- **Наслідки:** При видаленні автомобіля в системі можуть залишатися пов'язані історичні дані (напр. фінансові транзакції), але сам об'єкт ТЗ зникає з усіх активних списків та звітів.
- **Стан завантаження:** Під час виконання запиту кнопка змінює текст на "Видалення..." та блокується.
