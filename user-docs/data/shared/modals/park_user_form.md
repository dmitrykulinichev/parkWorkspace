# Модальне вікно: Додати/Редагувати користувача

{{SCRN:modal_park_user_form}}

> Використовується для створення нового користувача парку або редагування даних існуючого.

## Поля форми

| Поле | Тип | Обов'язкове | Опис / Технічні деталі |
| :--- | :--- | :---: | :--- |
| **Ім'я** | text | Так | Повне ім'я користувача. |
| **Email** | text | Так | Електронна пошта для входу та отримання сповіщень. |
| **Роль** | select | Так | Системна роль користувача (напр., Admin). Визначає базовий набір прав. |

## Логіка та валідація

- **Обов'язкові поля:** Всі поля форми є обов'язковими для заповнення.
- **Новий користувач:** При створенні нового користувача йому автоматично надсилається лист на вказаний Email з посиланням для активації та встановлення пароля.
- **Редагування:** При редагуванні Email не можна змінити, якщо користувач вже активований (залежить від реалізації бекенду, у формі поле доступне).
- **Обробка помилок:** Валідація унікальності Email виконується на стороні сервера.
