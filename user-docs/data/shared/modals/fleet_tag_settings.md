# Модальне вікно: Налаштування тегів

{{MODAL:modal_fleet_tag_settings}}

Вікно для управління словником кольорових міток (тегів), які використовуються для класифікації автомобілів у парку.

## Функціонал вікна

### 1. Список існуючих тегів
- **Колір:** Візуальний вибір кольору за допомогою вбудованого `input[type="color"]`.
- **Назва:** Текстове поле для зміни найменування тегу.
- **Індикація змін:** Якщо тег змінено, з'являється кнопка "Зберегти" (Save).
- **Видалення:** Кнопка "Trash" для повного видалення тегу з системи та всіх автомобілів.

### 2. Створення нового тегу
- Відкривається при кліку на кнопку "Додати новий тег".
- Дозволяє відразу вибрати колір та ввести назву.

## Логіка та особливості
- **Тип тегу:** Вікно працює виключно з тегами типу `vehicle`.
- **Вплив:** Видалення тегу в цьому вікні призведе до його автоматичного зникнення з карток усіх автомобілів, яким він був призначений.
- **Автофокус:** При додаванні нового тегу фокус автоматично переходить на поле вводу назви.
