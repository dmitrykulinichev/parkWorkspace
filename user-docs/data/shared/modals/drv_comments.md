# Модальне вікно: Коментарі до водія

{{SCRN:modal_drv_comments}}

> Вікно перегляду та додавання коментарів до профілю водія. Відкривається кліком на кнопку "+" у колонці "Коментарі" таблиці на сторінці `/drivers`. Редагування існуючого коментаря відкриває окреме вікно `DriverEditCommentModal`.

## Логіка та валідація

- **Режими вікна:** Перегляд списку коментарів та форма додавання нового.
- **Прив'язка:** Коментарі прив'язуються до сутності `Driver` через `commentableType` та `commentableId`.

## Технічна інформація

> Цей розділ призначений для розробників.
