# Модальне вікно: Додати/Редагувати файл парку

{{SCRN:modal_park_file_form}}

> Використовується для завантаження нових документів парку або редагування приміток та вартості існуючих.

## Поля форми

| Поле | Тип | Обов'язкове | Опис / Технічні деталі |
| :--- | :--- | :---: | :--- |
| **Файли** | file | Так | Один або декілька файлів документа. Використовує `FileManager`. |
| **Примітки** | textarea | Ні | Опис документа або додаткова інформація. |
| **Вартість (опціонально)** | number | Ні | Грошова сума, пов'язана з цим документом (напр., вартість страхування чи оренди). |

## Логіка та валідація

- **Обов'язковість файлів:** Форма не дозволить збереження, якщо не завантажено жодного файлу.
- **FileManager:** Дозволяє завантажувати нові файли та керувати вже прикріпленими.
- **Після збереження:** Список файлів на сторінці оновлюється автоматично.
