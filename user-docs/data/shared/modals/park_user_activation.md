# Модальне вікно: Посилання на активацію

{{SCRN:modal_park_user_activation}}

> Відображає пряме посилання на активацію аккаунта, якщо користувач не отримав автоматичного листа.

## Функціональні можливості

- **Перегляд посилання:** Відображає згенерований URL для завершення реєстрації.
- **Копіювання:** Кнопка "Скопіювати" для швидкого копіювання посилання в буфер обміну для ручної передачі користувачу.

## Логіка та валідація

- **Безпека:** Посилання містить унікальний токен і має обмежений термін дії.
- **Закриття:** Кнопка "Зрозуміло" закриває вікно після копіювання.
