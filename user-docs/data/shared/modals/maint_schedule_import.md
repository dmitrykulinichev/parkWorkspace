# Модальне вікно: Імпорт шаблонів ТО

{{MODAL:modal_maint_schedule_import}}

Інструмент для швидкого наповнення бази регламентів за допомогою попередньо налаштованих стандартних пакетів обслуговування.

## Функціонал
- **Вибір пакету:** Список готових наборів регламентів (наприклад, "Стандартний пакет для легкових авто", "Пакет для таксі").
- **Попередній перегляд:** Список робіт, які будуть додані в систему після імпорту.
- **Конфлікти:** Якщо регламент з такою назвою вже існує, система запропонує оновити його або пропустити.

## Технічна інформація

| Файл | Роль |
| :--- | :--- |
| `resources/js/components/modals/ImportSchedulesModal.jsx` | Компонент вікна імпорту. |
| `API: POST /api/v1/maintenance-schedules/import` | Ендпоінт для пакетного створення регламентів. |
