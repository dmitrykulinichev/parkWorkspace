# Модальне вікно: Здати автомобіль (Зняття водія)

{{MODAL:modal_ops_unassign_form}}

Спрощена форма для швидкої фіксації повернення автомобіля водієм.

## Вміст вікна
- **Інформаційний блок:** Відображає поточного водія та останній зафіксований пробіг авто.
- **Час здачі:** Фактична дата та час завершення роботи.
- **Пробіг (км):** Показники одометра на момент здачі авто. Має бути не меншим за початковий.
- **Причина здачі:** Короткий коментар про стан авто або причину повернення (напр. "Зміна завершена").

## Наслідки дії
1.  Активне призначення водія закривається.
2.  **Статус авто** змінюється на "Вільний".
3.  Оновлюється глобальний пробіг автомобіля в системі.

## Технічна інформація

| Файл | Роль |
