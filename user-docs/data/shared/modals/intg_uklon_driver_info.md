# Модальне вікно: Інформація про водія (Uklon)

{{SCRN:modal_intg_uklon_driver_info}}

> Вікно перегляду детальних даних водія з інтеграції Uklon (баланс, карма, статус, ліміти). Відкривається кліком на іконку Uklon у таблиці водіїв на сторінці `/drivers`.

## Логіка та валідація

- **Дані:** Завантажуються асинхронно через `uklonService.getDriver` при відкритті вікна.
- **Стани:** Відображає лоадер під час завантаження та повідомлення про помилку у разі невдачі (з можливістю повторного запиту).
- **Візуалізація:** Використовує динамічні кольори для статусів водія та Karma-показників.
