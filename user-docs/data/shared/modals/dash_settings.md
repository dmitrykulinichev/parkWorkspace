# Модальне вікно: Налаштування дашборду

{{SCRN:modal_dash_settings_form}}

> Дозволяє вмикати та вимикати окремі віджети головної панелі. Відкривається з кнопки-шестерінки у хедері сторінки `/`.

## Поля форми

| Поле | Тип | Обов'язкове | Опис / Технічні деталі |
| :--- | :--- | :---: | :--- |
| **Блок налаштувань (Onboarding)** | toggle | Ні | Картки швидкого налаштування системи |
| **Пульс парку** | toggle | Ні | Графік активності за 72 год |
| **Пульс Uklon API** | toggle | Ні | Графік активності Uklon API за 48 год |
| **Статистика замовлень Uklon** | toggle | Ні | Графік статусів замовлень Uklon |
| **Статуси автомобілів** | toggle | Ні | Зведення по статусах авто |
| **Відкриті звіти** | toggle | Ні | Список активних звернень водіїв |
| **Відкриті задачі** | toggle | Ні | Список невиконаних задач по авто |
| **Критичне ТО** | toggle | Ні | Авто, що потребують термінового ТО |
| **Проблеми з документами** | toggle | Ні | Прострочені страховки та ліцензії |
| **Огляд автопарку** | toggle | Ні | Загальна статистика та метрики |

## Логіка та валідація

- **Після збереження:** Зміни вступають у силу одразу — кожен `toggle` викликає `toggleWidget(id)`, стан зберігається в `localStorage` (`dashboard_settings_v1`).
- **Режими вікна:** Один режим — перегляд і редагування одночасно. Немає режиму "тільки перегляд".
- **Кнопка "Скинути":** Повертає всі налаштування до дефолтних (`DEFAULT_DASHBOARD_CONFIG` — всі віджети увімкнені).
