# Модальне вікно: Ручне списання боргу

{{SCRN:modal_manual_debt_payment_form}}

> Форма для внесення ручного платежу в рахунок погашення боргу водія. Відкривається з головної таблиці боргів.

## Поля форми

| Поле | Тип | Обов'язкове | Опис / Технічні деталі |
| :--- | :--- | :---: | :--- |
| **Сума списання** | number | Так | Сума, що списується в борг |
| **Дата списання** | date | Так | Дата операції |
| **Коментар** | textarea | Ні | Пояснення щодо платежу |

## Логіка та валідація

- **Валідація:** Сума списання не може перевищувати поточний залишок боргу (`remaining_amount`).
- **Після збереження:** Автоматичне оновлення прогресу боргу.
