# Модальне вікно: Призначення водія

{{MODAL:modal_ops_assignment_form}}

Це вікно призначене для закріплення водія за конкретним автомобілем на певний період. Воно також дозволяє завершувати існуючі призначення.

## Поля форми

| Поле | Опис | Технічні деталі |
| :--- | :--- | :--- |
| **Водій** | Вибір водія зі списку активних. | Обов'язкове. |
| **Автомобіль** | Вибір транспортного засобу. | Обов'язкове. |
| **Дата призначення** | Час початку використання авто водієм. | За замовчуванням — поточний. |
| **Початковий пробіг** | Показник одометра при отриманні авто. | Зазвичай підтягується останній відомий пробіг авто. |
| **Причина** | Текстове пояснення призначення. | Опціонально. |

### Секція "Завершення призначення" (лише редагування)
- **Toggle "Завершити":** Активує поля для фіксації здачі авто.
- **Дата завершення:** Час повернення авто в парк.
- **Кінцевий пробіг:** Одометр на момент здачі.
- **Причина завершення:** Коментар щодо стану авто при здачі.

## Логіка та конфлікти
- **Конфлікт (409):** Якщо водій вже призначений на інше авто або авто вже зайняте, система видасть попередження.
- **Примусове призначення:** У разі конфлікту з'являється кнопка "Примусово призначити", яка дозволяє менеджеру ігнорувати попередження (наприклад, для виправлення історичних даних).
