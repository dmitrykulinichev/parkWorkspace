# Модальне вікно: Створити / Редагувати призначення

{{SCRN:modal_ops_assignment_form}}

> Форма для призначення автомобіля водію або редагування існуючого призначення. Відкривається кнопкою "Створити призначення" на сторінці `/assignments` або через контекстне меню.

## Поля форми

| Поле | Тип | Обов'язкове | Опис / Технічні деталі |
| :--- | :--- | :---: | :--- |
| **Водій** | select | Так | Пошук та вибір водія з доступних |
| **Автомобіль** | select | Так | Вибір автомобіля з парку |
| **Дата призначення** | date | Так | Дата і час початку використання |
| **Примітка** | textarea | Ні | Додаткові відомості про призначення |

## Логіка та валідація

- **Обов'язкові поля:** Водій, Автомобіль, Дата призначення.
- **Обробка помилок:** Сервер повертає 409 Conflict, якщо авто вже зайняте іншим водієм у цей період.
- **Після збереження:** Автоматичне створення запису в журналі призначень, оновлення статусу авто (якщо застосовно).
