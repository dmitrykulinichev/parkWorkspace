# Модальне вікно: Підтвердження видалення

{{SCRN:modal_park_user_delete}}

> Запит на підтвердження безповоротного видалення користувача з парку.

## Логіка та валідація

- **Наслідки:** Користувач втрачає доступ до парку. Його історичні дії в журналах (logs) зберігаються, але посилаються на видалений об'єкт.
- **Обмеження:** Неможливо видалити самого себе.
