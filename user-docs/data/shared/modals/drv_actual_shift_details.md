# Модальне вікно: Деталі фактичної зміни

{{SCRN:modal_drv_actual_shift_details}}

> Вікно призначене для перегляду деталей фактично відпрацьованої водієм зміни. Відкривається при кліку на інформацію про фактичне призначення.

## Логіка та валідація

- **Режими вікна:** Відображення деталей, отриманих від системи фактичних призначень (наприклад, інтеграції).
- **Дані:** Відображає реальні часові мітки виходу на лінію та завершення роботи, а також примітки.

## Технічна інформація

| :--- | :--- |

| `resources/js/queries/useActualAssignments.js` | Запит даних про фактичні зміни. |
