# Модальне вікно: Форма графіку роботи

{{SCRN:modal_drv_schedule_slot_form}}

> Вікно призначене для додавання або редагування слоту в графіку роботи водія. Відкривається при кліку на клітинку графіку або кнопку додавання слоту.

## Поля форми

| Поле | Тип | Обов'язкове | Опис / Технічні деталі |
| :--- | :--- | :---: | :--- |
| **Автомобіль** | select | Так | Вибір авто з доступних у парку. |
| **Водій** | select | Так | Вибір водія з доступного пулу. |
| **Зміна** | select | Так | Вибір типу зміни (день, ніч тощо). |
| **Дата** | date | Так | Дата призначення слоту. |
| **Коментар** | text | Ні | Додаткова інформація до слоту. |

## Логіка та валідація

- **Обов'язкові поля:** Автомобіль, Водій, Зміна, Дата.
- **Залежності полів:** Вибір водія автоматично враховує його статус (доступність у вибрану дату).
- **Обробка помилок:** Сервер повертає помилку при накладанні слотів на одного водія або авто в той самий час.
- **Після збереження:** Автоматичне оновлення матриці графіку (`scheduleMatrix`).
