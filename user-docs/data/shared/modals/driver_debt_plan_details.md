# Модальне вікно: Деталі плану боргу

{{SCRN:modal_driver_debt_plan_details}}

> Вікно детального перегляду плану боргу водія, включаючи прогрес погашення та історію платежів.

## Функціональність

- **Перегляд:** Відображення загальних даних, поточного статусу та візуального прогрес-бару.
- **Історія платежів:** Таблиця з усіма проведеними списаннями (дата, сума, опис, хто виконав).
- **Редагування:** Можливість редагування окремого платежу (за наявності дозволів).
