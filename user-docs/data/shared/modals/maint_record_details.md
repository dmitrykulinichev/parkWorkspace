# Модальне вікно: Деталі заїзду на сервіс

{{MODAL:modal_maint_record_details}}

Це вікно надає повну інформацію про конкретний візит автомобіля на сервіс, включаючи детальний перелік робіт та прикріплені документи.

## Структура вікна

### 1. Header (Шапка)
- **ID Запису:** Унікальний номер у системі.
- **Автомобіль:** Посилання на ТЗ та його державний номер.
- **Дата:** Коли було виконано сервіс.
- **Статус:** Відображає, чи є цей запис активним.

### 2. Загальні дані заїзду
- **Пробіг:** Показник одометра на момент заїзду.
- **Загальна вартість:** Фінальна сума за всі роботи та запчастини.
- **Створив:** ПІБ користувача, який вніс дані.
- **Коментар:** Загальна примітка до всього візиту.

### 3. Список виконаних робіт
Детальна таблиця, що містить:
- **Тип та назва:** Конкретна робота (ТО або Ремонт).
- **Вартість:** Ціна за одиницю роботи/запчастини.
- **Файли:** Списки документів, прикріплених до конкретної роботи.
- **Коментар:** Деталі виконання кожного пункту.

### 4. Документи (Медіа-галерея)
Окремий блок для перегляду всіх завантажених файлів заїзду (чеки, акти, фото). Файли можна відкривати для перегляду або завантажувати.

## Доступні дії
- **Друк чеку:** Генерація друкованої версії звіту про сервіс (Receipt).
- **Редагувати:** Швидкий перехід до форми {{MODAL:maint_record_form}} для внесення змін.
- **Видалити:** Видалення всього заїзду та всіх пов'язаних робіт.
- **Закрити:** Повернення до журналу ТО.
