# Модальне вікно: Премія або Штраф

{{SCRN:modal_bonus_penalty_form}}

> Форма для створення нового запису про премію або штраф для водія, або редагування існуючого запису. Відкривається зі сторінки {{MODAL:bonuses_penalties}}.

## Поля форми

| Поле | Тип | Обов'язкове | Опис / Технічні деталі |
| :--- | :--- | :---: | :--- |
| **Тип** | select | Так | Оберіть "Премія" або "Штраф" |
| **Сума** | number | Так | Грошова сума у гривнях |
| **Водій** | select | Так | Оберіть водія зі списку |
| **Автомобіль** | select | Ні | Прив'язка до конкретного авто (опціонально) |
| **Дата та час** | date | Так | Дата і час події |
| **Опис / Причина** | textarea | Ні | Детальний опис для чого нараховано |

## Логіка та валідація

- **Обов'язкові поля:** Тип, Сума, Водій, Дата.
- **Режими вікна:** Підтримує як створення нового, так і редагування існуючого запису.
- **Після збереження:** Автоматично оновлюється список премій/штрафів на головній сторінці.
