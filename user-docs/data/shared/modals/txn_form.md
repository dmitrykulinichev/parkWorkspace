# Модальне вікно: Форма транзакції

> Вікно для додавання або редагування фінансової транзакції.

## Поля форми

| Поле | Тип | Обов'язкове | Опис / Технічні деталі |
| :--- | :--- | :---: | :--- |
| **Сума** | number | Так | Грошовий еквівалент транзакції. |
| **Тип** | select | Так | Дохід або витрата. |
| **Категорія** | select | Так | Категорія фінансової операції. |
| **Опис** | text | Ні | Деталі транзакції. |
| **Дата** | date | Так | Дата проведення операції. |

## Логіка та валідація

- **Обов'язкові поля:** Сума, Тип, Категорія, Дата.
