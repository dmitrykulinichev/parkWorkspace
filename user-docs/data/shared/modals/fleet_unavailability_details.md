# Модальне вікно: Деталі простою автомобіля

{{MODAL:modal_fleet_unavailability_details}}

Це вікно надає вичерпну інформацію про конкретний період недоступності автомобіля, включаючи причини, тривалість та відповідальних осіб.

## Структура даних
- **Автомобіль:** Повна назва, марка, модель та державний номер.
- **Тип та стан:** Категорія (Ремонт, ТО / Огляд, Конфіскований) та поточний статус (Активний/Закритий).
- **Хронологія:** Точний час початку та (якщо закрито) завершення періоду.
- **Тривалість:** Автоматично розрахований час перебування в недоступності (дні, години, хвилини).
- **Відповідальні:** ПІБ користувачів системи, які ініціювали відправку авто та закрили запис.
- **Причина та результат:** Повний текст коментарів, наданих на кожному етапі процесу.

## Призначення
Використовується менеджерами для глибокого аналізу історії ремонтів конкретного авто, перевірки ефективності роботи СТО або для з'ясування причин тривалого простою техніки.
