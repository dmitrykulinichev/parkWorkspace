# Модальне вікно: Управління правами (Permissions)

{{SCRN:modal_park_user_permissions}}

> Дозволяє детально налаштовувати індивідуальні дозволи для користувача, що доповнюють або обмежують його рольові права.

## Структура вікна

Вікно розділене на дві основні секції:
- **Дозвільні:** Прямі дозволи, які надаються користувачу.
- **Заборонні (deny:):** Дозволи, які явно забороняють певні дії, навіть якщо вони дозволені роллю.

## Функціональні можливості

- **Перемикання (Toggle):** Клік по пермішену вмикає або вимикає його. Права, успадковані від ролі, позначені та не можуть бути вимкнені індивідуально (потрібно змінювати роль).
- **Експорт:** Копіює список прямих пермішенів користувача у форматі JSON в буфер обміну.
- **Імпорт:** Дозволяє вставити JSON-масив пермішенів для швидкого синхронізації прав між користувачами.

## Логіка та валідація

- **Пріоритет ролі:** Якщо право надане роллю, воно відображається як активне з поміткою "від ролі".
- **Синхронізація:** При імпорті всі поточні прямі права користувача замінюються на нові з JSON-масиву.
