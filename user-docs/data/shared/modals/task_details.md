# Модальне вікно: Деталі задачі

{{SCRN:modal_task_details}}

> Вікно для перегляду повної інформації про задачу, включаючи пов'язані звіти про проблеми та історію виконання.

## Опис інтерфейсу

### Заголовок та Статус
- Назва задачі великим шрифтом.
- Бейджі статусу (Нова, В роботі, Завершена тощо) та пріоритету (Низький, Середній, Високий, Критичний).
- Позначка "Критична", якщо задача має критичний пріоритет.

### Опис
- Текст опису задачі зі збереженням форматування (переноси рядків).

### Пов'язані звіти про проблеми
- Відображається лише за наявності зв'язків.
- Загальна кількість звітів.
- Список карток звітів: заголовок, скорочений опис, дата створення.
- Кнопка швидкого переходу до перегляду конкретного звіту.

### Відповідальні та Час
- **Створив:** Користувач, який додав задачу.
- **Призначено:** Користувач-виконавець.
- **Створено:** Дата та час створення.
- **Завершено:** Дата завершення та ім'я користувача, який закрив задачу.
- **Час роботи:** Загальна кількість годин, витрачених на виконання.

### Автомобіль
- Картка автомобіля (держномер, назва), якщо задача прив'язана до ТЗ.

### Додаткова інформація
- Коментар виконавця, залишений при завершенні задачі.

## Управління (Actions)
- **Редагувати:** Відкриває {{MODAL:task_form}}.
- **Завершити:** Викликає діалог підтвердження завершення.
- **Видалити:** Викликає діалог підтвердження видалення.
