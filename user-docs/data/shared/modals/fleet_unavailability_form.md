# Модальне вікно: Відправити авто в недоступний стан

{{MODAL:modal_fleet_unavailability_form}}

Це вікно призначене для офіційної реєстрації виведення автомобіля з експлуатації.

## Поля форми

| Поле | Опис | Технічні деталі |
| :--- | :--- | :--- |
| **Автомобіль** | Вибір транспортного засобу. | Недоступно для зміни при редагуванні існуючого запису. |
| **Тип** | Категорія причини простою. | Ремонт, ТО / Огляд, Конфіскований. |
| **Причина** | Текстове пояснення. | Опціонально. Описує, що саме сталося. |
| **Дата початку** | Коли авто перестало бути доступним. | За замовчуванням — поточний час. |
| **Планова дата повернення** | Орієнтовний час завершення. | Опціонально. Допомагає планувати графік змін. |

## Логіка та обмеження
- **Захист від дублювання:** Система видасть помилку (Конфлікт 409), якщо ви спробуєте відправити в недоступність авто, яке вже там знаходиться.
- **Вплив на систему:** Відразу після збереження статус автомобіля змінюється на "Ремонт" (або інший відповідний), що робить його невидимим для нових призначень у графіку.
- **Редагування:** При редагуванні додається поле **"Нотатки при закритті"**, що дозволяє змінити коментар завершення без відкриття спеціального вікна закриття.
