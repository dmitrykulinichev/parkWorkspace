# Модальне вікно: Повернути авто в роботу (Закрити запис)

{{MODAL:modal_fleet_unavailability_close}}

Вікно для фіксації завершення періоду недоступності та повернення транспортного засобу до активної експлуатації.

## Вміст вікна
- **Інформаційна панель:** Нагадування, яке саме авто повертається в роботу.
- **Дата закриття:** Фактична дата та час завершення робіт або повернення авто (за замовчуванням — поточний час).
- **Нотатки при закритті:** Поле для опису результатів (наприклад: "Ремонт виконано, замінено щеплення").

## Наслідки дії
1.  Запис про недоступність отримує статус "Закритий".
2.  Розраховується фінальна тривалість простою.
3.  **Статус автомобіля** автоматично змінюється на "Вільний", що дозволяє знову призначати на нього водіїв та планувати зміни у графіку.

## Технічна інформація

| Файл | Роль |
| :--- | :--- |
| `resources/js/components/modals/CloseUnavailabilityModal.jsx` | Компонент вікна закриття. |
| `resources/js/queries/vehicleUnavailabilities.js` | Мутація `useCloseVehicleUnavailability`. |
| `resources/js/hooks/useVehicleUnavailabilitiesPage.js` | Обробник `handleCloseSuccess`, що оновлює дані на сторінці. |
| `API: PUT /api/v1/vehicle-unavailabilities/{id}/close` | Ендпоінт для фіксації завершення простою. |
