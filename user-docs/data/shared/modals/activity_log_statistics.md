# Модальне вікно: Статистика журналу подій

{{SCRN:modal_activity_log_statistics}}

> Аналітична панель, що відображає активність користувачів та системи за вибраний період.

## Огляд
Вікно дозволяє переглядати статистику роботи системи, активність користувачів та технічні параметри (пристрої) для виявлення підозрілої або інтенсивної діяльності.

## Функціонал

- **Вибір періоду:** 7, 30, 90 днів або 1 рік.
- **Основні метрики:** Всього дій, активних користувачів, критичних подій, середнє значення дій на день.
- **Деталізація:**
  - Розподіл по типах дій.
  - Розподіл по пристроях (Desktop, Mobile, Tablet).
  - Список найактивніших користувачів.
