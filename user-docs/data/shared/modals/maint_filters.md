# Модальне вікно: Фільтри журналу ТО

{{SCRN:modal_maint_filters}}

Спеціалізований інструмент для швидкого пошуку сервісних подій за набором технічних та часових критеріїв.

## Доступні фільтри

| Параметр | Опис |
| :--- | :--- |
| **Період** | Вибір часового проміжку (Від/До) за датою виконання робіт. |
| **Автомобіль** | Фільтрація записів по конкретній одиниці техніки. |
| **Тип робіт** | Вибір між "ТО" (регламент) та "Ремонт" (позапланово). |
| **Конкретна робота** | Пошук за назвою регламентної роботи (напр. "Заміна ГРМ"). |
| **Хто створив** | Фільтрація за менеджером або механіком, який вніс запис. |
| **Вартість** | Пошук записів у певному діапазоні цін. |

## Логіка використання
- **Комбінування:** Можна одночасно шукати, наприклад, "всі ремонти для Toyota Camry за березень".
- **Скидання:** Можливість миттєво очистити всі фільтри.
- **Активні чіпи:** Обрані параметри відображаються над таблицею журналу для візуального контролю та швидкого видалення окремих умов.
