# Модальне вікно: Видалення новини

{{SCRN:modal_landing_news_delete}}

> Модальне вікно для підтвердження видалення новини.

## Логіка та валідація

- **Підтвердження:** Вікно відображає назву новини, яку планується видалити.
- **Після видалення:** Новина миттєво зникає зі списку, запит на сервер видаляє її з бази даних.
