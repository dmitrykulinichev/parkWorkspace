# Модальне вікно: Створити/Редагувати план розстрочки

{{SCRN:modal_driver_debt_plan_form}}

> Форма для налаштування умов боргу водія (загальна сума, розмір платежу, графік). Відкривається зі сторінки {{MODAL:driver_debt_plans}}.

## Поля форми

| Поле | Тип | Обов'язкове | Опис / Технічні деталі |
| :--- | :--- | :---: | :--- |
| **Водій** | select | Так | Оберіть водія (заблоковано при редагуванні) |
| **Автомобіль** | select | Ні | Прив'язка до авто |
| **Загальна сума боргу** | number | Так | Повна сума боргу |
| **Сума платежу** | number | Так | Сума одного періодичного платежу |
| **Частота списання** | select | Так | Щодня, щотижня, щомісяця або вручну |
| **Дата початку** | date | Так | Дата першого платежу |
| **Опис / Причина** | textarea | Ні | Деталізація причин виникнення боргу |

## Логіка та валідація

- **Обов'язкові поля:** Водій, Загальна сума, Сума платежу, Частота, Дата початку.
- **Обмеження:** Суми повинні бути > 0.
- **Статуси:** Після створення план за замовчуванням має статус `active`.
