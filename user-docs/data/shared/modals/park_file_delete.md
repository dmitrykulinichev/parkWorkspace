# Модальне вікно: Підтвердження видалення документа

{{SCRN:modal_park_file_delete}}

> Запит на підтвердження видалення документа парку.

## Логіка та валідація

- **Наслідки:** Видалення документа призводить до автоматичного видалення всіх пов'язаних фізичних файлів та фінансових транзакцій (якщо вони були створені на основі поля "Вартість").
- **Дія незворотна:** Після підтвердження дані неможливо відновити.
