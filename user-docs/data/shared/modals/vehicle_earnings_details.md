# Модальне вікно: Деталі заробітку автомобіля

{{SCRN:modal_vehicle_earnings_details}}

> Вікно відображає розширену фінансову статистику для обраного автомобіля за вказаний період.

## Поля форми

Дане вікно не є формою для введення, а використовується для відображення деталізованих даних.

| Поле | Опис |
| :--- | :--- |
| **Період** | Відображається період, за який вибрано дані. |
| **Фінансові метрики** | Деталізований перелік доходів (Uklon, ручні), комісій, витрат та чистого прибутку. |

## Логіка та валідація

- **Джерело даних:** Вікно отримує `vehicle` (об'єкт авто) та `filters` (дати) від батьківського компонента.
- **Оновлення:** Дані оновлюються при зміні вибраного авто або періоду фільтрації на сторінці.
