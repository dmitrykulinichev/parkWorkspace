# Модальне вікно: Фільтри файлів

{{SCRN:modal_park_file_filters}}

> Призначене для фільтрації списку документів парку за типом, датою або іншими атрибутами.

## Поля форми

| Поле | Тип | Обов'язкове | Опис / Технічні деталі |
| :--- | :--- | :---: | :--- |
| **Період** | date range | Ні | Фільтрація за датою створення документа. |
| **Створив** | select | Ні | Фільтрація за користувачем, який додав документ. |

## Логіка та валідація

- **Скидання фільтрів:** Кнопка "Очистити" повертає список до повного відображення.
- **Застосування:** Фільтри застосовуються до реєстру після натискання кнопки підтвердження.
