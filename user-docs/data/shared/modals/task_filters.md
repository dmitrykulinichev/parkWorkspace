# Модальне вікно: Фільтри задач

{{SCRN:modal_task_filters}}

> Вікно для розширеної фільтрації списку задач. Дозволяє звузити вибірку за статусом, пріоритетом, виконавцем та іншими параметрами.

## Поля форми

| Поле | Тип | Обов'язкове | Опис / Технічні деталі |
| :--- | :--- | :---: | :--- |
| **Автомобіль** | multiselect | Ні | Вибір одного або декількох авто. |
| **Статус** | multiselect | Ні | Фільтрація за станом (Нова, В роботі тощо). |
| **Пріоритет** | multiselect | Ні | Фільтрація за рівнем важливості. |
| **Виконавець** | multiselect | Ні | Фільтрація за користувачем, якому призначена задача. |
| **Створив** | multiselect | Ні | Фільтрація за автором задачі. |
| **Дата створення** | date_range | Ні | Фільтрація задач, створених у вказаний проміжок часу. |

## Логіка та валідація

- **Динамічні опції:** Список доступних фільтрів та їх опцій завантажується з ендпоінту `/api/v1/tasks/options`.
- **Скидання:** Кнопка "Скинути всі" очищає поточні налаштування фільтрів.
- **Синхронізація:** Вибрані значення автоматично відображаються як чіпи в хедері сторінки.
