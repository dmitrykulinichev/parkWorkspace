# Recorder Instructions

> Інструкції для захоплення скріншотів — вручну або через Katalon Recorder.
> Заповнюється автоматично під час Фази 2 (планування скріншотів).
> Кожен блок описує мінімальний набір дій щоб відтворити потрібний стан екрана.

---

## Формат блоку

```
## [id скріншота]

- **URL:** `/маршрут`
- **Кроки:**
  1. Дія 1
  2. Дія 2
- **Тригер:** текст кнопки або CSS-селектор (для модалок і вкладок)
```

Для простих сторінок де єдиний крок — відкрити URL — нумеровані кроки не потрібні.

## page_fleet_list_main

- **URL:** `/vehicles`
- **Кроки:** Дочекатися завантаження таблиці (мають бути дані).

## page_fleet_list_empty

- **URL:** `/vehicles`
- **Кроки:** Відкрити з порожнім автопарком (видалити або приховати всі авто).

## page_fleet_list_mobile

- **URL:** `/vehicles`
- **Кроки:** Встановити viewport < 768px, дочекатися адаптивного рендеру.

## modal_fleet_veh_form

- **URL:** `/vehicles`
- **Кроки:**
  1. Дочекатися завантаження сторінки
  2. Натиснути кнопку "Додати авто" в хедері
- **Тригер:** кнопка з текстом "Додати авто"

## modal_fleet_veh_details

- **URL:** `/vehicles`
- **Кроки:**
  1. Дочекатися завантаження таблиці
  2. Клікнути на рядок будь-якого авто
- **Тригер:** клік по рядку таблиці (row click)

## modal_fleet_veh_filters

- **URL:** `/vehicles`
- **Кроки:**
  1. Натиснути кнопку фільтрів (іконка або текст "Фільтри")
- **Тригер:** кнопка відкриття фільтрів у хедері

## modal_intg_uklon_vehicle_details

- **URL:** `/vehicles`
- **Кроки:**
  1. Дочекатися завантаження таблиці (авто мають бути зв'язані з Uklon)
  2. Знайти рядок авто з активною Uklon-іконкою (не сірою)
  3. Клікнути на іконку Uklon в колонці "Інфо"
- **Тригер:** клік на Uklon-іконку (`img[alt="Uklon"]`) в колонці "Інфо", тільки якщо `vehicle.uklon_id` присутній

## tab_rep_reports_maint

- **URL:** `/reports`
- **Кроки:**
  1. Дочекатися завантаження сторінки
  2. Клікнути на вкладку "ТО"
- **Тригер:** вкладка з текстом "ТО" / "Технічне обслуговування"
