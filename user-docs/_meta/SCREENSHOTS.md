# Стандарт найменування скріншотів (Isolated)

Ця копія правил створена для самодостатності папки `user-docs`.

## Формула назви
`[тип]_[модуль]_[об'єкт]_[дія/стан].png`

---

## 1. Типи (Prefix)
- `page_` — ціла сторінка.
- `modal_` — модальне вікно.
- `widget_` — окремий функціональний блок.
- `mobile_` — мобільний інтерфейс.
- `print_` — вигляд для друку/PDF.

## 2. Модулі (Scope)
- `dashboard` — головна панель.
- `fleet` — автопарк.
- `drivers` — водії.
- `fin` — фінанси.
- `sch` — графік (schedule).
- `maint` — технічне обслуговування.
- `uklon` — інтеграція з агрегатором.
- `tg` — Telegram бот.
- `admin` — налаштування парку.

## 3. Об'єкти та Дії
- `list` — таблиця, реєстр.
- `form` / `create` / `edit` — форми вводу.
- `details` / `view` — детальна інформація.
- `empty` — без даних.
