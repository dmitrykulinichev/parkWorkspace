# Стандарт найменування скріншотів (Isolated)

## Формула назви
`[Тип]_[Модуль]_[Об'єкт]_[Стан].png`

---

## 1. Типи (Пріоритет створення)
1. `page_` — **Full Page** (Вся сторінка). Найвищий пріоритет.
2. `modal_` — **Modal** (Модальне вікно). Середній пріоритет.
3. `sec_` — **Section** (Секція, віджет або специфічний елемент). Низький пріоритет.

## 2. Модулі (Scope)
- `dash` — Dashboard.
- `fleet` — Автомобілі/Автопарк.
- `maint` — ТО та ремонти.
- `docs` — Документи.
- `rep` — Звіти.
- `ops` — Операційна діяльність (призначення тощо).
- `drv` — Водії.

## 3. Приклади
- `page_fleet_list_main.png` (Вся сторінка списку авто)
- `modal_fleet_vehicle_create.png` (Модалка створення авто)
- `sec_fleet_stats_cards.png` (Секція статистики вгорі сторінки)
