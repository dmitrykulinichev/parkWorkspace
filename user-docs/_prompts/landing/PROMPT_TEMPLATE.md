# Заготовка промпту — Landing Docs

Скопіюй і заповни перед запуском.

---

```
Прочитай інструкції:
- `workspace/user-docs/_prompts/landing/LANDING_TASK.md`
- `workspace/user-docs/_prompts/landing/LANDING_STYLE.md`

Задача: згенерувати сторінку документації лендінгу на основі:
`workspace/user-docs/data/[шлях до файлу]`
```

---

## Де взяти шлях до файлу

Дивись `workspace/user-docs/_meta/INDEX.md` — там список всіх задокументованих сторінок.

## Якщо хочеш інший стиль

Створи новий файл поруч з `LANDING_STYLE.md` (наприклад, `LANDING_STYLE_FORMAL.md`) і вкажи його замість основного.
